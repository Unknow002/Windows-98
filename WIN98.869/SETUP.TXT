ΓΝΩΣΤΑ ΠΙΘΑΝΑ ΠΡΟΒΛΗΜΑΤΑ ΕΓΚΑΤΑΣΤΑΣΗΣ


ΓΡΗΓΟΡΕΣ ΣΥΜΒΟΥΛΕΣ ΓΙΑ ΕΓΚΑΤΑΣΤΑΣΗ ΧΩΡΙΣ ΣΦΑΛΜΑΤΑ
Απενεργοποιήστε όλα τα προγράμματα αντιμετώπισης ιών
που εκτελούνται στο σύστημά σας. 
Εάν αυτά τα βοηθητικά προγράμματα εκτελούνται 
στη διάρκεια της Εγκατάστασης, το σύστημά σας μπορεί 
να σταματήσει να ανταποκρίνεται. 

ΣΗΜΕΙΩΣΗ: Ορισμένα συστήματα διαθέτουν ενσωματωμένες  
δυνατότητες αντιμετώπισης ιών. Εάν αυτή η επιλογή παραμείνει
ενεργοποιημένη στις ρυθμίσεις BIOS/CMOS, ενδέχεται να λάβετε
προειδοποίηση σχετικά με "δραστηριότητες ιών" ή με αλλαγές
στη "Βασική εγγραφή εκκίνησης". Πρέπει να αφήσετε να γίνουν
αυτές οι αλλαγές, ώστε το πρόγραμμα της Εγκατάστασης να ολοκληρωθεί
με επιτυχία. Ανατρέξτε στην τεκμηρίωση του λογισμικού αντιμετώπισης
ιών για περισσότερες πληροφορίες.

Εκτελέστε την Εξέταση Δίσκων, για να ελέγξετε και να διορθώσετε
οποιαδήποτε προβλήματα έχετε στο σκληρό δίσκο (ή στους σκληρούς
δίσκους).

Κλείστε όλα τα προγράμματα που εκτελούνται.
Αυτό περιλαμβάνει την απενεργοποίηση οποιασδήποτε προφύλαξης οθόνης,
των ρυθμίσεων "Προηγμένης διαχείρισης μνήμης" και άλλων
προγραμμάτων που μπορεί να κάνουν το πρόγραμμα της Εγκατάστασης
να μην ανταποκρίνεται. Για να εξασφαλίσετε καθαρή αναβάθμιση
από τα Windows 95, πριν από την εκτέλεση του προγράμματος
Εγκατάστασης πιέστε το συνδυασμό πλήκτρων
CTRL+ALT+DEL και το κουμπί "Τέλος εργασίας" για κάθε 
πρόγραμμα που βρίσκεται σε λειτουργία εκτός από το SYSTRAY
και το Explorer. Ανατρέξτε στην ενότητα "Εκτέλεση καθαρής εκκίνησης" 
για περισσότερες πληροφορίες.

ΓΕΝΙΚΑ ΘΕΜΑΤΑ ΕΓΚΑΤΑΣΤΑΣΗΣ
Εάν έχετε Number Nine Imagine 128 Display 
Adapter ή STB Velocity 128 3D AGP (Nvidia 
Riva 128), θα πρέπει να εκτελέσετε το πρόγραμμα
Εγκατάστασης από το MS-DOS ή 
να αλλάξετε το πρόγραμμα οδήγησης οθόνης σε VGA.

Εκδόσεις αναβάθμισης ή πλήρους εγκατάστασης των Windows 98. 
Εάν έχετε την έκδοση Αναβάθμισης των Windows 98, 
το πρόγραμμα Εγκατάστασης θα προσπαθήσει να βρει ένα
κατάλληλο πρόγραμμα αναβάθμισης στο σύστημά σας.
Εάν το πρόγραμμα Εγκατάστασης δεν βρει προηγούμενη
έκδοση των Windows, θα σας ζητηθεί να εισαγάγετε το
προηγούμενο αποθηκευτικό σας μέσο για έλεγχο της δυνατότητας
για αναβάθμιση.

Απαιτήσεις χώρου στο δίσκο για τα Windows 98. 
Επειδή υπεισέρχονται πολλοί παράγοντες 
στον υπολογισμό του ελεύθερου χώρου στο δίσκο
που απαιτείται για τα Windows 98, τα παρακάτω
μεγέθη είναι απλές εκτιμήσεις, βασισμένες
σε τυπικές εγκαταστάσεις των Windows 98.

Τυπική αναβάθμιση από Windows 95: απαιτεί περίπου 
  195 MB ελεύθερου χώρου στον σκληρό δίσκο, αλλά μπορεί να κυμαίνεται 
  από 120 MB μέχρι 295 MB, ανάλογα με τις παραμέτρους 
  του συστήματός σας και τις επιλογές που αποφασίζετε να εγκαταστήσετε.
Πλήρης εγκατάσταση των Windows 98 σε μονάδα δίσκου FAT16: απαιτεί 
  225 MB ελεύθερου χώρου στον σκληρό δίσκο, αλλά μπορεί να κυμαίνεται μεταξύ 
  165 ΜΒ και 355 MB, ανάλογα με τις παραμέτρους του συστήματός σας και 
  τις επιλογές που ορίσατε.
Πλήρης εγκατάσταση των Windows 98 σε μονάδα δίσκου FAT32: απαιτεί 
  175 MB ελεύθερου χώρου στον σκληρό δίσκο, αλλά μπορεί να κυμαίνεται μεταξύ 
  140 MB και 255 MB, ανάλογα με τις παραμέτρους του συστήματος και 
  τις επιλογές που ορίσατε.

Επίσης, εάν εγκαθιστάτε τα Windows 98 σε μονάδα δίσκου
διαφορετική από το C, το πρόγραμμα Εγκατάστασης μπορεί να απαιτήσει μέχρι 25 MB 
ελεύθερου χώρου στον δίσκο στη μονάδα C για τα αρχεία συστήματος
και καταγραφής που δημιουργούνται κατά την Εγκατάσταση.

Κατάργηση της εγκατάστασης: Εάν θέλετε να δημιουργήσετε
ένα αντίγραφο ασφαλείας των 
Windows 95 πριν την αναβάθμιση, μπορείτε να ορίσετε την επιλογή "Αποθήκευση συστήματος"
κατά την Εγκατάσταση. Αυτό θα σας επιτρέψει
να καταργήσετε την εγκατάσταση των Windows 98 σε περίπτωση
που αντιμετωπίσετε δυσκολίες. Ωστόσο, σε ορισμένες περιπτώσεις
δεν μπορείτε να καταργήσετε την εγκατάσταση:

* Η τρέχουσα εγκατάσταση των Windows έχει γίνει σε 
  συμπιεσμένη μονάδα δίσκου.
* Κάνετε εγκατάσταση σε νέο κατάλογο ή κάνετε 
  καθαρή εγκατάσταση χωρίς να υπάρχει προηγούμενη έκδοση διαθέσιμη.
* Εκτελείτε έκδοση του MS-DOS προγενέστερη 
  της έκδοσης 5.0.

ΔΗΜΙΟΥΡΓΗΣΤΕ ΜΙΑ ΝΕΑ ΔΙΣΚΕΤΑ ΕΚΚΙΝΗΣΗΣ! Λόγω των αλλαγών 
στους πυρήνες πραγματικής και προστατευμένης λειτουργίας 
για την υποστήριξη του FAT32, οι δισκέτες εκκίνησης των Windows 98 
δεν είναι συμβατές με προηγούμενες εκδόσεις των Windows. 
Κατά συνέπεια, όταν εγκαθιστάτε τα Windows 98 για πρώτη φορά, 
φροντίστε να δημιουργήσετε μια νέα δισκέτα εκκίνησης, 
ΑΚΟΜΗ ΚΑΙ ΑΝ ΔΕΝ ΣΚΟΠΕΥΕΤΕ ΝΑ ΧΡΗΣΙΜΟΠΟΙΗΣΕΤΕ ΤΟ FAT32.

Διαχείριση Προγραμμάτων των Windows 3.x.
Η Διαχείριση Προγραμμάτων δεν υποστηρίζεται πλέον στα Windows 98. 
Η Διαχείριση Προγραμμάτων (Progman.exe) παραμένει στο σύστημα 
για λόγους αντιμετώπισης προβλημάτων, αλλά ΔΕΝ θα περιέχει 
καμία ομάδα. Επιπλέον, αν κάνετε αναβάθμιση από 
Windows 95, τα υπάρχοντα αρχεία .grp θα καταργηθούν. 
Έχει διαπιστωθεί ότι αυτά τα αρχεία προκαλούν ορισμένα σφάλματα 
κατά την εγκατάσταση των Windows 98. Εάν κάνετε αναβάθμιση από Windows 3.x, 
τα παλιά αρχεία .grp θα παραμείνουν στο σύστημα και 
η Διαχείριση Προγραμμάτων θα έχει ακόμη κάποιες δυνατότητες. 
Θα πρέπει να δημιουργήσετε αντίγραφα ασφαλείας των υπαρχόντων αρχείων progman.ini
και *.grp πριν να κάνετε αναβάθμιση σε Windows 98, εάν σκοπεύετε να
χρησιμοποιήσετε τη Διαχείριση Προγραμμάτων.

ΕΓΚΑΤΑΣΤΑΣΗ ΤΩΝ WINDOWS 98 ΑΠΟ ΤΟ MS-DOS
Εάν ο σκληρός σας δίσκος είναι καθαρός ή νέος ή 
εάν αντιμετωπίζετε δυσκολίες κατά την εκτέλεση του προγράμματος Εγκατάστασης
από προηγούμενη έκδοση των Windows, ίσως χρειαστεί
να εκτελέσετε το πρόγραμμα Εγκατάστασης των Windows 98 
από το MS-DOS. Παρόλο που η εγκατάσταση από το MS-DOS είναι 
χαρακτηριστικά η πιο αργή μέθοδος εγκατάστασης, είναι 
συχνά η ασφαλέστερη και θα πρέπει να τη χρησιμοποιείτε 
όταν οι άλλοι τύποι εγκατάστασης αποτυγχάνουν.

Πλήκτρα συντόμευσης για εκκίνηση από το MS-DOS:
Υπάρχουν πολλοί τρόποι ασφαλούς εκκίνησης του συστήματός σας 
από τη γραμμή εντολών του MS-DOS. Ο ευκολότερος τρόπος είναι
να χρησιμοποιήσετε τα παρακάτω πλήκτρα συντόμευσης:

* Windows 98
Κρατήστε πατημένο το πλήκτρο CTRL, ενώ κάνετε εκκίνηση του υπολογιστή σας. 
Αυτό θα σας μεταφέρει κατευθείαν στο Μενού εκκίνησης των Windows 98 
(το πλήκτρο F8 συνεχίζει να λειτουργεί, αλλά στα Windows 98 δεν εμφανίζεται μήνυμα
Starting Windows 98, συνεπώς είναι δύσκολο να γνωρίζετε πότε ακριβώς να το πιέσετε).

* Windows 95
Πιέστε το πλήκτρο F8 μόλις εμφανιστεί το μήνυμα Starting Windows 95. 
Αυτό θα σας μεταφέρει στο Μενού εκκίνησης των Windows 95.

* MS-DOS 6.x
Πιέστε το πλήκτρο F8 μόλις εμφανιστεί το μήνυμα Starting MS-DOS. Αυτό 
σας επιτρέπει να επιλέξετε εσείς ποια προγράμματα οδήγησης
θα φορτωθούν ή θα παρακαμφθούν από τα αρχεία συστήματος. 

* Προγράμματα οδήγησης CD-ROM πραγματικής κατάστασης λειτουργίας
Θα πρέπει να είναι φορτωμένα τα προγράμματα οδήγησης CD-ROM 
πραγματικής κατάστασης λειτουργίας, ώστε να έχετε πρόσβαση στο CD
των Windows 98. Εάν είχατε εκτελέσει παλαιότερα το πρόγραμμα
Εγκατάστασης των Windows 98 και είχατε δημιουργήσει δισκέτα εκκίνησης,
μπορείτε να χρησιμοποιήσετε τα προγράμματα οδήγησης του CD-ROM που
περιέχονται σε αυτήν τη δισκέτα. Εάν δεν έχετε δισκέτα εκκίνησης,
θα χρειαστεί να εκτελέσετε το πρόγραμμα Εγκατάστασης που συνόδευε το υλικό
του CD-ROM που αγοράσατε.

Αφού αποκτήσετε πρόσβαση στη μονάδα του CD-ROM, 
μπορείτε να μεταβείτε στη μονάδα που περιέχει το CD 
των Windows 98 και να πληκτρολογήσετε την εντολή: SETUP. 
Το πρόγραμμα της Εγκατάστασης θα πρέπει κανονικά να συνεχίσει τη λειτουργία του.


Συμβουλές για την εγκατάσταση προγραμμάτων οδήγησης
CD-ROM πραγματικής κατάστασης λειτουργίας:

Εάν εκτελείτε Windows 95:
Εάν εκτελείτε Windows 95, μπορεί να έχετε ήδη φορτωμένο
ένα τμήμα των προγραμμάτων οδήγησης του CD-ROM. 
Εάν μπορέσετε να κάνετε επανεκκίνηση του υπολογιστή σε 
λειτουργία MS-DOS και να αποκτήσετε πρόσβαση 
στη μονάδα του CD-ROM, προσπαθήστε να κάνετε τα παρακάτω:

* Ξεκινήστε πάλι τον υπολογιστή και πιέστε το πλήκτρο F8 
  μόλις εμφανιστεί το μήνυμα Starting Windows 95.
* Κάντε την επιλογή Command Prompt Only.
* Στη γραμμή εντολών C:\ πληκτρολογήστε: DosStart.bat.

Τώρα θα πρέπει να έχετε πρόσβαση στη μονάδα του CD-ROM.

Απώλεια πρόσβασης στη μονάδα του CD-ROM κατά την Εγκατάσταση:
Εάν στη διάρκεια της Εγκατάστασης των Windows 98 δεν έχετε πλέον
πρόσβαση στο CD-ROM, προσπαθήστε να κάνετε τα παρακάτω:

* Ξεκινήστε πάλι τον υπολογιστή και πιέστε το πλήκτρο F8 μόλις εμφανιστεί
  το μήνυμα Starting Windows 95, και στη συνέχεια κάντε την 
  επιλογή Command Prompt Only. Εάν εκτελείτε MS-DOS, 
  κάντε εκκίνηση κατευθείαν στη γραμμή εντολών.
* Επεξεργαστείτε το αρχείο Autoexec.bat 
  πληκτρολογώντας την εντολή: Edit Autoexec.bat
* Διαγράψτε το κείμενο Rem by Windows 98 Setup μπροστά από 
  τη γραμμή που περιέχει την παραπομπή στο Mscdex.exe.
* Βγείτε από την Επεξεργασία πληκτρολογώντας ALT-F-X 
  και αποθηκεύστε το αρχείο, όταν σας ζητηθεί.
* Ξεκινήστε πάλι τον υπολογιστή. Είτε θα συνεχίσει μόνη της η Εγκατάσταση ή 
  θα πρέπει να εκτελέσετε ξανά το πρόγραμμα Εγκατάστασης, επιλέγοντας 
  την εντολή Safe Recovery, εάν σας ζητηθεί.

Εκτέλεση του προγράμματος της Εγκατάστασης των Windows 98 από
τον σκληρό δίσκο:
Εάν αντιγράψετε όλα τα αρχεία του προγράμματος της Εγκατάστασης στον
σκληρό δίσκο και στη συνέχεια κάνετε την εγκατάσταση από
τον σκληρό δίσκο, μπορείτε να αποφύγετε τα περισσότερα
σφάλματα που σχετίζονται με θέματα αντιγραφής αρχείων και 
Ι/Ο δίσκου. Μπορείτε να επιλέξετε ώστε να μην παραμένουν τα προγράμματα οδήγησης
CD-ROM στη μνήμη και να ελευθερώσετε συμβατική μνήμη,
για να ελαχιστοποιήσετε τα σφάλματα ανεπαρκούς μνήμης αυτού του
τύπου εγκατάστασης. Για να αντιγράψετε τοπικά τα αρχεία
του προγράμματος Εγκατάστασης:

Από τα Windows 95:
* Ελευθερώστε επιπλέον 120 MB χώρο στον σκληρό δίσκο συν 
  το χώρο που θα απαιτήσει η Εγκατάσταση. Συνήθως, η Εγκατάσταση απαιτεί
  195 MB για την αναβάθμιση από τα Windows 95.
* Δημιουργήστε έναν προσωρινό φάκελο για να αποθηκεύσετε τα αρχεία 
  του προγράμματος Εγκατάστασης σε αυτήν τη μονάδα δίσκου.
* Αντιγράψτε τα περιεχόμενα του φακέλου Win98 του CD των 
  Windows 98 στον προσωρινό φάκελο που μόλις δημιουργήσατε. 
  Επίσης θα πρέπει να αντιγράψετε τους υποφακέλους των Win98, αλλά αυτό 
  δεν είναι απαραίτητο αν δεν έχετε πολύ χώρο στο δίσκο.
* Ξεκινήστε πάλι τον υπολογιστή. Πιέστε το πλήκτρο F8 μόλις εμφανιστεί το 
  μήνυμα Starting Windows 95 και κάντε την επιλογή Safe Mode Command Prompt Only. 
* Τώρα, μεταβείτε στον προσωρινό φάκελο που περιέχει τα αρχεία του
  προγράμματος Εγκατάστασης των Windows 98 και πληκτρολογήστε: SETUP.

Από το MS-DOS:
* Βεβαιωθείτε ότι έχετε πρόσβαση στη μονάδα του CD-ROM.
  Δείτε παραπάνω για περισσότερες πληροφορίες.
* Ελευθερώστε επιπλέον 120 MB χώρο στον σκληρό δίσκο συν 
  το χώρο που θα απαιτήσει η Εγκατάσταση. Συνήθως, η Εγκατάσταση απαιτεί
  195 MB για την αναβάθμιση από τα Windows 95.
* Δημιουργήστε έναν προσωρινό φάκελο στη μονάδα με πολύ 
  ελεύθερο χώρο για την αποθήκευση των αρχείων του προγράμματος της
  Εγκατάστασης. Για να  δημιουργήσετε έναν προσωρινό κατάλογο, μεταβείτε
  στο γράμμα αυτής της μονάδας δίσκου και πληκτρολογήστε την εντολή: MD Win98tmp.
* Τώρα, μεταβείτε στη μονάδα του CD-ROM των Windows 98 και στον κατάλογο 
  Win98 πληκτρολογώντας: <γράμμα μονάδας>\Win98.
* Στη συνέχεια αντιγράψτε το αρχείο του προγράμματος Εγκατάστασης των Windows 98 
  στον προσωρινό κατάλογο που μόλις δημιουργήσατε, πληκτρολογώντας: 
  Copy *.* <γράμμα μονάδας>\Win98tmp
* Αφού αντιγραφούν όλα τα αρχεία, ξεκινήστε πάλι το σύστημά σας 
  και κάντε καθαρή εκκίνηση παρακάμπτοντας τα αρχεία εκκίνησης. 
  Ανατρέξτε στην ενότητα "Εκτέλεση καθαρής εκκίνησης" για περισσότερες 
  πληροφορίες.
* Μεταβείτε στον προσωρινό κατάλογο στον οποίο μόλις αντιγράψατε 
  τα αρχεία και κάντε εκκίνηση του προγράμματος Εγκατάστασης 
  πληκτρολογώντας την εντολή: SETUP.

Εκτέλεση καθαρής εκκίνησης:
Προγράμματα οδήγησης συσκευών, βοηθητικά προγράμματα ή άλλα προγράμματα
άλλων κατασκευαστών μπορεί να εμποδίσουν την επιτυχή εγκατάσταση. 
Κάνοντας καθαρή εκκίνηση του συστήματος μπορείτε να διορθώσετε
πολλά από αυτά τα θέματα. Μπορείτε να εκτελέσετε καθαρή εκκίνηση 
με τους ακόλουθους τρόπους:

Χρήση δισκέτας για την εκκίνηση του υπολογιστή:
* Εκκίνηση από δισκέτα εκκίνησης των Windows 98. Αυτή η δισκέτα επιτρέπει 
  την επιλογή φόρτωσης με ή χωρίς τα προγράμματα οδήγησης CD-ROM 
  και αποτελεί καθαρό περιβάλλον για την εκτέλεση του προγράμματος Εγκατάστασης.
* Εκκίνηση από δισκέτα εκκίνησης προηγούμενων Windows 95 ή MS-DOS. 
  Αυτή η εκκίνηση δεν παρέχει πρόσβαση στα προγράμματα οδήγησης CD-ROM, αλλά 
  μπορεί να χρησιμοποιηθεί εάν αντιγράψετε τα αρχεία του προγράμματος Εγκατάστασης 
  στον σκληρό δίσκο όπως περιγράφεται παραπάνω.

Windows 95 Safe Mode Command Prompt Only:
* Κάντε εκκίνηση του συστήματος και κρατήστε πατημένο το πλήκτρο F8 
  μόλις εμφανιστεί το μήνυμα Starting Windows 95.
* Επιλέξτε Safe Mode Command Prompt Only. Ούτε αυτή η επιλογή 
  παρέχει πρόσβαση στη μονάδα του CD-ROM, αλλά μπορεί να 
  χρησιμοποιηθεί εάν τα αρχεία του προγράμματος Εγκατάστασης αντιγραφούν στον σκληρό 
  δίσκο, όπως περιγράφεται παραπάνω.

Εκκίνηση των Windows 98 βήμα προς βήμα:
Εάν θέλετε να κάνετε μη αυτόματη φόρτωση ορισμένων προγραμμάτων οδήγησης,
κάντε τα ακόλουθα:
* Κάντε εκκίνηση του συστήματος και κρατήστε πατημένο το πλήκτρο F8 
  μόλις εμφανιστεί το μήνυμα Starting Windows 95.
* Ενεργοποιήστε την επιλογή Step by Step.
* Τώρα, επιλέξτε YES μόνο για τις συσκευές που θέλετε να φορτωθούν. Στις 
  περισσότερες περιπτώσεις, θα πρέπει να επιλέξετε YES για το Himem.sys

Καθαρή εκκίνηση από Windows 95/MS-DOS με περισσότερη μνήμη:
Μπορείτε να αυξήσετε τη διαθέσιμη μνήμη κάνοντας 
τις παρακάτω τροποποιήσεις στο αρχείο Config.sys. 
Επίσης μπορείτε να κάνετε αυτές τις αλλαγές και στη δισκέτα εκκίνησης. 
ΣΗΜΕΙΩΣΗ: Αυτά είναι τα μόνα προγράμματα οδήγησης που πρέπει να φορτώσετε.

Device=Himem.sys
Device=EMM386.exe noems
Dos=high,umb
Device=drvspace.sys /move 
	(Προαιρετικά - μόνο εάν χρησιμοποιείτε τη Συμπίεση Δίσκων)

Δημιουργία δισκέτας εκκίνησης των Windows 98:
Εάν η Εγκατάσταση των Windows 98 αποτύχει μετά την αντιγραφή των περισσότερων αρχείων 
στον σκληρό δίσκο, ίσως να μπορείτε να δημιουργήσετε δισκέτα εκκίνησης 
χρησιμοποιώντας το βοηθητικό πρόγραμμα bootdisk.bat.
* Κάντε εκκίνηση σε γραμμή εντολών του MS-DOS.
* Αλλάξτε κατάλογο και μεταβείτε στον κατάλογο Windows\command.
* Εκτελέστε το πρόγραμμα Bootdisk.bat, το οποίο θα σας ζητήσει να 
  δημιουργήσετε δισκέτα εκκίνησης.

Αυτή η δισκέτα περιέχει προγράμματα οδήγησης CD-ROM πραγματικής λειτουργίας
και γενικής χρήσης, τα οποία μπορεί να χρησιμεύσουν όταν θα εκτελέσετε ξανά
το πρόγραμμα της Εγκατάστασης.

ΛΟΓΙΣΜΙΚΟ ΑΝΤΙΜΕΤΩΠΙΣΗΣ ΙΩΝ
Βεβαιωθείτε ότι δεν λειτουργεί κανένα πρόγραμμα αντιμετώπισης ιών 
κατά την εγκατάσταση των Windows 98. Εάν το πρόγραμμα παραμένει στη μνήμη 
(TSR), καταργήστε οποιαδήποτε παραπομπή σε αυτό 
στα αρχεία Autoexec.bat, Config.sys και Win.ini.

Εάν το BIOS έχει ενσωματωμένη προστασία από ιούς, 
απενεργοποιήστε την πριν να εκτελέσετε το πρόγραμμα Εγκατάστασης. 
Για να την απενεργοποιήσετε, πρέπει να χρησιμοποιήσετε το πρόγραμμα
εγκατάστασης CMOS για το BIOS. Για περισσότερες πληροφορίες, 
ανατρέξτε στην τεκμηρίωση του υπολογιστή σας.

Ανατρέξτε στις παρακάτω σημειώσεις για συγκεκριμένα προγράμματα αντιμετώπισης
ιών.

Προστασία από ιούς με ενεργοποιημένο CMOS/BIOS:
Ορισμένα συστήματα έχουν ενσωματωμένη προστασία από ιούς. 
Εάν παραμείνει ενεργοποιημένη η προστασία από ιούς, μπορεί
να λάβετε προειδοποιητικά μηνύματα  για "Δραστηριότητες ιών" ή για 
"Αλλαγή της βασικής εγγραφής εκκίνησης". Πρέπει να αφήσετε
να γίνουν αυτές οι αλλαγές. Εάν επιλέξετε επαναφορά των
προηγούμενων ρυθμίσεων, το σύστημά σας μπορεί να
μην κάνει πια εκκίνηση.

Norton AntiVirus:
Εάν είναι εγκατεστημένο το πρόγραμμα Norton AntiVirus, 
μπορεί να δείτε την ακόλουθη προειδοποίηση στο τέλος της
αρχικής διαδικασίας αντιγραφής αρχείων: 

	Application Wininst0.400\Suwin.exe is 
	attempting to update the Master Boot Record

Θα πρέπει να επιλέξετε Continue (C), για να ολοκληρωθεί η Εγκατάσταση 
σωστά. Εάν δεν επιτρέψετε να γίνουν αυτές οι αλλαγές, 
το πρόγραμμα Εγκατάστασης ίσως σταματήσει να ανταποκρίνεται.

Dr. Solomon's AntiVirus:
Εάν έχετε σε λειτουργία το βοηθητικό πρόγραμμα Dr. Solomon's AntiVirus, 
μπορεί να εμφανιστεί μια γαλάζια οθόνη με ένα μήνυμα σφάλματος ανεπανόρθωτης
εξαίρεσης στο Ios.vxd, ενώ προσπαθείτε να δημιουργήσετε μια δισκέτα εκκίνησης
στη διάρκεια της Εγκατάστασης. Θα πρέπει να κάνετε κλικ στο κουμπί "’κυρο"
στην οθόνη της δισκέτας εκκίνησης, όταν η γραμμή προόδου
βρίσκεται στο 20%. Έτσι, θα συνεχιστεί η λειτουργία του 
προγράμματος Εγκατάστασης. Αναζητήστε μια ενημέρωση του λογισμικού 
Dr. Solomon's AntiVirus στην τοποθεσία της εταιρείας
στο Web, για να επιλύσετε αυτό το θέμα.

Εντοπισμός προβλημάτων κατά την Εγκατάσταση των Windows 98
Η έκδοση της Εξέτασης Δίσκων που εκτελείται κατά την Εγκατάσταση
των Windows 98 ελέγχει μόνο για σφάλματα. Δεν τα διορθώνει.
Εάν προκύψουν ζητήματα, το πρόγραμμα της Εγκατάστασης δεν μπορεί
να συνεχίσει αν δεν διορθωθούν. 
Για να διορθώσετε αυτά τα προβλήματα, κλείστε την Εγκατάσταση
και εκτελέστε την Εξέταση Δίσκων από τα Windows 95 ή το MS-DOS.
Περισσότερες πληροφορίες σχετικά με τη χρήση της Εξέτασης 
Δίσκων για την επίλυση αυτών των θεμάτων θα βρείτε στη συνέχεια.

Διόρθωση προβλημάτων του σκληρού δίσκου:
Εάν, κατά την Εγκατάσταση, εμφανιστεί ένα μήνυμα που σας λέει ότι
πρέπει να εκτελέσετε την Εξέταση Δίσκων για να διορθώσετε
τα προβλήματα του σκληρού δίσκου, ακολουθήστε την εξής
διαδικασία.

Εάν κάνετε εγκατάσταση των Windows 98 από το MS-DOS ή από
προηγούμενη έκδοση των Windows, όπως τα Windows 3.1:

1. Κλείστε τα Windows.

2. Εάν κάνετε εγκατάσταση από δισκέτες, τοποθετήστε
   τη Δισκέτα Εγκατάστασης 1 στη μονάδα δισκέτας 
   και στη συνέχεια πληκτρολογήστε τα ακόλουθα 
   στη γραμμή εντολών:

	a:scandisk.exe /all

   όπου a είναι η μονάδα που περιέχει τη δισκέτα των Windows.

3. Εάν κάνετε εγκατάσταση από CD, τοποθετήστε το CD και 
   στη συνέχεια πληκτρολογήστε τα ακόλουθα:

        d:\win98\scandisk.exe /all

   όπου d είναι η μονάδα που περιέχει το CD.

4. Ακολουθήστε τις οδηγίες στην οθόνη σας και διορθώστε οποιαδήποτε 
   προβλήματα εντοπίσει η Εξέταση Δίσκων.

5. Κάντε εκκίνηση των Windows και στη συνέχεια εκτελέστε 
   ξανά το πρόγραμμα της Εγκατάσταση.

Εάν εγκαθιστάτε τα Windows 98 επάνω σε μια προηγούμενη έκδοση 
των Windows 98 ή σε Windows 95:

1. Κλείστε την Εγκατάσταση.

2. Στο μενού "Έναρξη", τοποθετήστε τον δείκτη του ποντικιού στην επιλογή
   "Προγράμματα", κατόπιν στην επιλογή "Βοηθήματα", στη συνέχεια στην επιλογή
   "Εργαλεία συστήματος" και κάντε κλικ στην επιλογή "Εξέταση Δίσκων".

3. Ελέγξτε τους σκληρούς δίσκους και όλες τις μονάδες υποδοχής 
   για σφάλματα και διορθώστε οποιαδήποτε προβλήματα εντοπιστούν. Φροντίστε 
   να κάνετε πλήρη σάρωση επιφάνειας σε όλες τις μονάδες δίσκου, 
   διαφορετικά το πρόγραμμα Εγκατάστασης μπορεί να εντοπίσει πάλι σφάλματα.

Σφάλματα κατά την εκτέλεση της Εξέτασης Δίσκων:
Σε ορισμένες περιπτώσεις, η Εξέταση Δίσκων ενδέχεται
να μην μπορεί να διορθώσει ένα θέμα ή να δημιουργεί σφάλματα. 

Έχετε σε λειτουργία τη Συμπίεση Δίσκων 3:
Εάν η Συμπίεση Δίσκων 3 είναι εγκατεστημένη
στο σύστημά σας, ίσως να μην επαρκεί η συμβατική μνήμη του υπολογιστή σας.
Για να ελευθερώσετε μνήμη, μπορείτε να
να κάνετε τα παρακάτω:

* Εάν βρίσκεστε σε MS-DOS 6.x, μπορείτε να εκτελέσετε το αρχείο Memmaker.exe, 
  για να ελευθερώσετε αρκετή μνήμη, ώστε να ολοκληρωθεί η Εξέταση Δίσκων. 
* Ανατρέξτε στην ενότητα ΕΓΚΑΤΑΣΤΑΣΗ ΤΩΝ WINDOWS 98 ΑΠΟ ΤΟ MS-DOS 
  σχετικά με το πώς θα εκτελέσετε καθαρή εκκίνηση με περισσότερη μνήμη.
* Ελέγξτε τις μονάδες δίσκου ενώ εκτελείτε τα Windows 95.

Εάν συνεχίζετε να μην έχετε αρκετή μνήμη ή εάν αντιμετωπίζετε άλλες δυσκολίες,
ενώ η Εγκατάσταση εκτελεί την Εξέταση Δίσκων, μπορείτε 
να παρακάμψετε την Εξέταση Δίσκων κατά την Εγκατάσταση εκτελώντας
το πρόγραμμα Εγκατάστασης με την επιλογή /IS. Για να το κάνετε αυτό,
πληκτρολογήστε την ακόλουθη εντολή:

	setup /is

ΣΗΜΕΙΩΣΗ: Η παράκαμψη της Εξέτασης Δίσκων δεν προτείνεται. 
Εάν κάνετε παράκαμψη, μπορεί να προκύψουν ζητήματα με τον
σκληρό δίσκο, τα οποία θα εμποδίσουν την εγκατάσταση ή τη
σωστή εκτέλεση των Windows 98.  

ΚΑΤΑΡΓΗΣΗ ΤΩΝ WINDOWS 98
Αποθήκευση των αρχείων συστήματος:
Η Εγκατάσταση των Windows 98 παρέχει στους χρήστες την επιλογή 
δημιουργίας αντιγράφων ασφαλείας της προηγούμενης έκδοσης των Windows,
σε περίπτωση που χρειαστεί να καταργηθεί η εγκατάσταση των Windows 98  
αργότερα. Για να ενεργοποιήσετε αυτήν την επιλογή, 
πρέπει να ενεργοποιήσετε την επιλογή "Αποθήκευση συστήματος",
όταν σας ζητηθεί κατά την Εγκατάσταση. Σε αυτήν την περίπτωση,
το πρόγραμμα Εγκατάστασης θα δημιουργήσει τα ακόλουθα κρυφά
αρχεία που είναι απαραίτητα για την κατάργηση της εγκατάστασης των Windows 98:
  * Winundo.dat
  * Winundo.ini
  * Winlfn.ini

ΣΗΜΕΙΩΣΗ: Τυχόν διαγραφή αυτών των αρχείων θα εμποδίσει
την κατάργηση της εγκατάστασης των Windows 98.

Εάν ισχύει ένα από τα παρακάτω, δεν θα μπορέσετε 
να καταργήσετε την εγκατάσταση των Windows 98 και η Εγκατάσταση 
δεν θα σας ζητήσει να κάνετε "Αποθήκευση συστήματος":

* Η τρέχουσα εγκατάσταση των Windows βρίσκεται σε συμπιεσμένη 
  μονάδα δίσκου.
* Κάνετε εγκατάσταση σε νέο κατάλογο ή καθαρή 
  εγκατάσταση χωρίς να υπάρχει προηγούμενη έκδοση διαθέσιμη.
* Εκτελείτε μια έκδοση του MS-DOS προγενέστερη της έκδοσης 5.0.

ΣΗΜΕΙΩΣΗ: Τα αρχεία που είναι απαραίτητα για την
κατάργηση των Windows 98 πρέπει να αποθηκευτούν
σε τοπικό σκληρό δίσκο. Δεν μπορείτε να τα αποθηκεύσετε 
σε δίσκο δικτύου ή σε δισκέτα. Εάν έχετε δύο
ή περισσότερες μονάδες δίσκου με αρκετό ελεύθερο
χώρο, μπορείτε να επιλέξετε τη μονάδα στην οποία
θα αποθηκεύσετε τις πληροφορίες για την κατάργηση της εγκατάστασης.

Επίσης, υπάρχουν διάφορες ενέργειες που μπορεί να εμποδίσουν 
την κατάργηση της εγκατάστασης των Windows 98, αφού ολοκληρωθεί η Εγκατάσταση
των Windows 98. Στη λίστα που ακολουθεί περιγράφονται τα στοιχεία που θα
καταργήσουν τις πληροφορίες για την κατάργηση της εγκατάστασης από το σύστημά σας:

* Μετατροπή του σκληρού δίσκου σε FAT32
* Συμπίεση του σκληρού δίσκου με τη Συμπίεση Δίσκων

ΣΗΜΕΙΩΣΗ: Αυτά τα βοηθητικά προγράμματα θα πρέπει
να σας ειδοποιήσουν, πριν να εκτελέσουν τη μετατροπή ή τη συμπίεση,
ότι οι πληροφορίες για την κατάργηση της εγκατάστασης θα χαθούν.

Κατάργηση των Windows 98:
Για να καταργήσετε τα Windows 98 και να επαναφέρετε πλήρως το σύστημά σας 
στις προηγούμενες εκδόσεις του MS-DOS και των Windows 3.x ή 
των Windows 95:

1. Κάντε κλικ στο κουμπί "Έναρξη", τοποθετήστε το δείκτη του ποντικιού 
   στην επιλογή "Ρυθμίσεις" και στη συνέχεια κάντε κλικ στην επιλογή Πίνακας 
   Ελέγχου.
2. Κάντε διπλό κλικ στο εικονίδιο "Προσθαφαίρεση προγραμμάτων".
3. Στην καρτέλα "Εγκατάσταση/κατάργηση", κάντε κλικ στην 
   επιλογή "Κατάργηση της εγκατάστασης των Windows 98" και μετά κάντε κλικ 
   στο κουμπί "Προσθαφαίρεση".

Ή, εάν αντιμετωπίζετε δυσκολίες με την εκκίνηση των Windows 98, 
χρησιμοποιήστε τη δισκέτα εγκατάστασης για να ξεκινήσετε τον υπολογιστή σας
και στη συνέχεια εκτελέστε το αρχείο UNINSTAL από τη δισκέτα εκκίνησης.

ΣΗΜΕΙΩΣΗ: Το UNINSTAL πρέπει να τερματίσει τα Windows 98. Εάν 
ο υπολογιστής σας εμφανίσει σφάλμα κατά τον τερματισμό, ξεκινήστε πάλι
τον υπολογιστή σας και πατήστε το πλήκτρο F8 μόλις εμφανιστεί το μήνυμα Starting 
Windows 98. Στη συνέχεια, κάντε κλικ στην επιλογή Command Prompt Only 
και εκτελέστε το αρχείο UNINSTAL από τη γραμμή εντολών.

Εάν το πρόγραμμα Εγκατάστασης δεν ολοκληρώθηκε με επιτυχία και θέλετε να 
επαναφέρετε τις προηγούμενες εκδόσεις του MS-DOS και των Windows 3.x 
ή των Windows 95, μπορείτε να εκτελέσετε το αρχείο UNINSTAL από τον κατάλογο 
\Windows\Command στον σκληρό δίσκο ή από τη δισκέτα εκκίνησης.

Εάν δεν αποθηκεύσατε τα αρχεία σας στη μονάδα C, αλλά σε κάποια άλλη μονάδα, 
μπορείτε να χρησιμοποιήσετε την επιλογή /w, για να καθορίσετε τη μονάδα 
όπου βρίσκονται τα αρχεία. Για παράδειγμα:

        uninstal /w e:

όπου e: είναι η μονάδα που περιέχει τα προηγούμενα αρχεία συστήματος.

Εάν εκτελείτε Windows 98 και θέλετε να καταργήσετε τα 
αρχεία της Κατάργησης εγκατάστασης για να ελευθερώσετε χώρο στο δίσκο, 
ακολουθήστε την εξής διαδικασία:

1. Κάντε κλικ στο κουμπί "Έναρξη", τοποθετήστε το δείκτη του ποντικιού 
   στην επιλογή "Ρυθμίσεις" και στη συνέχεια κάντε κλικ στην επιλογή 
   Πίνακας Ελέγχου.
2. Κάντε διπλό κλικ στο εικονίδιο "Προσθαφαίρεση προγραμμάτων".
3. Στην καρτέλα "Εγκατάσταση/κατάργηση", κάντε κλικ στην επιλογή "Παλιά 
   αρχεία συστήματος Windows 3.x/MS-DOS" και στη συνέχεια κάντε κλικ 
   στο κουμπί "Κατάργηση". Ή κάντε κλικ στην επιλογή "Κατάργηση 
   των αρχείων συστήματος των Windows 95" (Πληροφορίες κατάργησης της εγκατάστασης).

Τώρα δεν μπορείτε πια να καταργήσετε τα Windows 98.

ΕΑΝ ΕΧΕΤΕ ΣΥΜΠΙΕΣΜΕΝΟ ΔΙΣΚΟ 
Εάν έχετε συμπιεσμένο δίσκο,
μπορεί να εμφανιστεί ένα μήνυμα ότι δεν υπάρχει αρκετός χώρος στο
διαμέρισμα της μονάδας υποδοχής του συμπιεσμένου δίσκου.
Η Εγκατάσταση μπορεί να χρειαστεί να αντιγράψει ορισμένα αρχεία
στη μονάδα του δίσκου εκκίνησης, τη μονάδα υποδοχής
για τη μονάδα δίσκου εκκίνησης ή τη μονάδα υποδοχής
για τη μονάδα δίσκου των Windows. Εάν εμφανιστεί αυτό το μήνυμα,
ελευθερώστε χώρο στη συγκεκριμένη μονάδα και στη συνέχεια
εκτελέστε ξανά το πρόγραμμα Εγκατάστασης. Προσπαθήστε να κάνετε
ένα από τα ακόλουθα:

* Εγκαταστήστε τα Windows σε μη συμπιεσμένο δίσκο, εάν είναι δυνατό.

* Διαγράψτε οποιαδήποτε αρχεία δεν είναι απαραίτητα από το διαμέρισμα της μονάδας υποδοχής.

* Εάν χρησιμοποιείτε Windows 3.1 και έχετε μόνιμο 
  αρχείο μετάθεσης, προσπαθήστε να μειώσετε το μέγεθός του. Στον Πίνακα Ελέγχου, 
  κάντε διπλό κλικ στο εικονίδιο 386 Enhanced και στη συνέχεια κάντε κλικ 
  στην επιλογή "Εικονική μνήμη". Τροποποιήστε το μέγεθος του αρχείου μετάθεσης.

* Χρησιμοποιήστε το λογισμικό συμπίεσης δίσκων για να ελευθερώσετε χώρο 
  στη μονάδα υποδοχής για τη συμπιεσμένη μονάδα δίσκου. Εάν 
  συμπιέσατε το δίσκο σας χρησιμοποιώντας τη "Συμπίεση Δίσκων" 
  ή το DoubleSpace, ακολουθήστε την εξής διαδικασία:

  1. Κλείστε τα Windows.
  2. Εκτελέστε το αρχείο Drvspace.exe ή το αρχείο Dblspace.exe.
  3. Επιλέξτε τη συμπιεσμένη μονάδα δίσκου στη μονάδα υποδοχής της οποίας θέλετε 
     να ελευθερώσετε χώρο.
  4. Στο μενού Drive, κάντε κλικ στην εντολή Change Size και ρυθμίστε τον 
     ελεύθερο χώρο όπως χρειάζεται.

  Εάν συμπιέσατε τη μονάδα δίσκου χρησιμοποιώντας τη Συμπίεση Δίσκων των Windows 95, 
  ή τη Συμπίεση Δίσκων 3 από το Plus!, ακολουθήστε την εξής διαδικασία:

  1. Εκκινήστε τα Windows
  2. Επιλέξτε την εντολή "Συμπίεση Δίσκων" επιλέγοντας
     Έναρξη/Προγράμματα/Βοηθήματα/Εργαλεία Συστήματος/Συμπίεση Δίσκων.
  3. Επιλέξτε τη συμπιεσμένη μονάδα στη μονάδα υποδοχής της οποίας θέλετε 
     να ελευθερώσετε χώρο.
  4. Από το μενού "Μονάδα δίσκου", επιλέξτε την εντολή "Ρύθμιση ελεύθερου χώρου" 
     και στη συνέχεια ρυθμίστε τον ελεύθερο χώρο όπως χρειάζεται.

Εάν χρησιμοποιήσατε άλλο λογισμικό συμπίεσης, όπως το Stacker, 
συμβουλευτείτε την τεκμηρίωση του λογισμικού.

ΣΗΜΕΙΩΣΗ: Μπορεί να παρατηρήσετε ασυμφωνία μεταξύ του ελεύθερου 
χώρου που αναφέρει το πρόγραμμα Εγκατάστασης και το χώρο που εσείς πιστεύετε 
ότι είναι διαθέσιμος στη μονάδα υποδοχής. Τα Windows χρησιμοποιούν λίγο χώρο 
για τη δημιουργία ενός αρχείου μετάθεσης. Αυτός ο χώρος μπορεί να μην εμφανίζεται 
εκχωρημένος, όταν δεν εκτελούνται τα Windows. 

ΣΗΜΕΙΩΣΗ: Εάν δημιουργήσετε δισκέτα εκκίνησης κατά την Εγκατάσταση, 
βεβαιωθείτε ότι δεν χρησιμοποιείτε συμπιεσμένο δίσκο για τη δισκέτα εκκίνησης.

ΕΓΚΑΤΑΣΤΑΣΗ ΤΩΝ WINDOWS 98 ΣΕ ΣΥΣΤΗΜΑ 
ΠΟΥ ΧΡΗΣΙΜΟΠΟΙΕΙ WINDOWS NT 
Δεν μπορείτε να εγκαταστήσετε τα Windows 98 σε οποιαδήποτε έκδοση των 
Windows NT, αλλά τα δύο λειτουργικά συστήματα μπορούν να συνυπάρχουν στο ίδιο σύστημα. 
Ωστόσο, για λόγους συμβατότητας, προτείνεται 
να τα εγκαταστήσετε σε ξεχωριστό σκληρό 
δίσκο ή διαμέρισμα. Εάν τα Windows NT είναι ήδη εγκατεστημένα, 
το πρόγραμμα Εγκατάστασης των Windows 98 θα προστεθεί στο μενού
εκκίνησης των Windows NT, ώστε ο χρήστης να έχει τη δυνατότητα επιλεκτικής
εκκίνησης μεταξύ Windows 98 και Windows NT.

Εάν δεν είναι δυνατή η εκκίνηση των Windows NT, θα πρέπει να
κάνετε εκκίνηση από τις δισκέτες ανάκτησης (Recovery disks) των Windows NT
και να ενεργοποιήσετε την επιλογή Repair, για να επαναφέρετε
τα αρχεία εκκίνησης των Windows NT.

Κατά την εγκατάσταση των Windows 98 σε ένα σύστημα με μονάδες δίσκου 
που δημιουργήθηκαν με τα Windows NT, μπορεί να εμφανιστεί το ακόλουθο 
μήνυμα σφάλματος:

"Η Εγκατάσταση εντόπισε ότι ο σκληρός σας δίσκος έχει ένα διαμέρισμα  
FAT συμπλεγμάτων 64Κ. Επειδή η Εξέταση Δίσκων δεν λειτουργεί
σε δίσκους με αυτό το μέγεθος συμπλέγματος, δεν είναι δυνατή
η συνέχιση της Εγκατάστασης. Για να συνεχίσετε την Εγκατάσταση, 
πρέπει να ξανακάνετε διαμερισμό του σκληρού σας δίσκου 
σε σύστημα αρχείων FAT με μέγεθος συμπλέγματος έως 
32K και να ξεκινήσετε πάλι την Εγκατάσταση."

Εάν εκτελέσετε την Εγκατάσταση με την παράμετρο /is (π.χ., Setup /is), 
θα παρακάμψετε την Εξέταση Δίσκων και θα αποφύγετε αυτό το θέμα.

ΕΓΚΑΤΑΣΤΑΣΗ ΤΩΝ WINDOWS 98 ΣΕ ΣΥΣΤΗΜΑ 
ΠΟΥ ΧΡΗΣΙΜΟΠΟΙΕΙ OS/2
Η Εγκατάσταση δεν εκτελείται σε σύστημα OS/2. Πρέπει να ξεκινήσετε το MS-DOS 
και να εκτελέσετε την Εγκατάσταση από τη γραμμή εντολών του MS-DOS.

ΣΗΜΕΙΩΣΗ: Εάν κάνετε αναβάθμιση από OS/2 σε διαμέρισμα HPFS, 
πρέπει να έχετε τη δισκέτα 1 του OS/2 έτοιμη κατά την Εγκατάσταση.

Εάν ξεκινήσετε το MS-DOS από δισκέτα και στη συνέχεια εκτελέσετε 
το πρόγραμμα Εγκατάστασης, δεν θα μπορείτε να ξεκινήσετε το OS/2 αφού 
εγκατασταθούν τα Windows 98. Πρέπει να διαγράψετε τα αρχεία 
Autoexec.bat και Config.sys που χρησιμοποιεί το OS/2, πριν να
εκτελέσετε το πρόγραμμα Εγκατάστασης των Windows 98.

Εάν θέλετε να καταργήσετε το OS/2 από τον υπολογιστή σας, αφού 
έχετε εγκαταστήσει τα Windows 98:

1. Δημιουργήστε αντίγραφα ασφαλείας των αρχείων δεδομένων που θέλετε να διατηρήσετε 
   σε μια δισκέτα ή σε δίσκο δικτύου.
2. Διαγράψτε τα αρχεία από κάθε κατάλογο και υποκατάλογο   
   του OS/2 και στη συνέχεια διαγράψτε τους καταλόγους του OS/2.
3. Στο εικονίδιο "Ο Υπολογιστής μου" ή στην Εξερεύνηση των Windows, 
   κάντε κλικ στο μενού "Προβολή", κάντε κλικ στην εντολή "Επιλογές" 
   και ενεργοποιήστε την επιλογή "Εμφάνιση όλων των αρχείων". 
   Στη συνέχεια, διαγράψτε τα ακόλουθα αρχεία του OS/2:

	Ea data.sf
	OS2ldr.msg
	OS2krnl
	OS2boot
	Wp data.sf

ΜΗΝΥΜΑΤΑ ΣΦΑΛΜΑΤΟΣ ΕΓΚΑΤΑΣΤΑΣΗΣ
Αυτή η ενότητα περιγράφει συγκεκριμένα μηνύματα που μπορεί 
να εμφανιστούν κατά την Εγκατάσταση και παρέχει πληροφορίες σχετικά 
με το τι πρέπει να κάνετε στη συνέχεια.

Μήνυμα: "Προηγούμενη έκδοση του αρχείου Setupx.dll"

Η Εγκατάσταση εντοπίζει ότι χρησιμοποιείται παλαιότερη
έκδοση του αρχείου SetupX.dll ή του αρχείου NetDi.dll. 
Αυτό μπορεί να συμβαίνει επειδή υπάρχουν ανοιχτές εφαρμογές
στον υπολογιστή σας. Εάν θέλετε να αντικαταστήσει
η Εγκατάσταση το υπάρχον αρχείο και να κάνει αυτόματα
επανεκκίνηση του συστήματός σας, κάντε κλικ στο κουμπί "OK". 
Διαφορετικά, για να διατηρήσετε την υπάρχουσα έκδοση 
του αρχείου SetupX.dll και του αρχείου NetDi.dll, κάντε κλικ 
στο κουμπί "’κυρο", για να κλείσετε την Εγκατάσταση, 
κλείστε οποιεσδήποτε ανοιχτές εφαρμογές και στη συνέχεια
εκτελέστε ξανά το πρόγραμμα Εγκατάστασης.

Αυτό το σφάλμα μπορεί να οφείλεται σε διάφορα προγράμματα:

* In Control Tools της Diamond
* Προγράμματα οδήγησης κάρτας ήχου Ensonics AudioPCI
* Προγράμματα οδήγησης κάρτας ήχου Aztech
* Βοηθητικά προγράμματα ρύθμισης παραμέτρων για τα modem Hayes
* Ορισμένα βοηθητικά προγράμματα συστήματος εκτύπωσης των Windows 
  (Σειρά Canon BJ)

Εάν η Εγκατάσταση των Windows 98 συνεχίζει να εμφανίζει αυτό το σφάλμα μετά 
την επανεκκίνηση, θα πρέπει να εκτελέσετε το πρόγραμμα Εγκατάστασης από τη
γραμμή εντολών του MS-DOS. Δείτε την ενότητα "ΕΓΚΑΤΑΣΤΑΣΗ ΤΩΝ 
WINDOWS 98 ΑΠΟ ΤΟ MS-DOS" για περισσότερες πληροφορίες.

Μήνυμα SU0018
"Η Εγκατάσταση δεν είναι δυνατό να δημιουργήσει αρχεία στη μονάδα εκκίνησης και 
να εγκαταστήσει τα Windows 98. Ο ριζικός κατάλογος του δίσκου
εκκίνησης ίσως περιέχει πάρα πολλά αρχεία ή άλλαξε 
η αντιστοίχιση του γράμματός του."

Ο ριζικός φάκελος μιας μονάδας δίσκου διατηρεί μέχρι 512 καταχωρίσεις 
(αρχεία ή φακέλους). Αυτό το μήνυμα δηλώνει ότι η Εγκατάσταση 
εντόπισε πάρα πολλές καταχωρίσεις καταλόγου στον ριζικό
φάκελο του υπολογιστή σας και η Εγκατάσταση δεν μπορεί να
δημιουργήσει τα αρχεία που χρειάζεται για την εγκατάσταση των 
Windows 98. Μετακινήστε ή διαγράψτε ορισμένα αρχεία από τον
ριζικό φάκελο της μονάδας και στη συνέχεια εκτελέστε ξανά 
το πρόγραμμα της Εγκατάστασης.

Αυτό το μήνυμα σφάλματος μπορεί να εμφανιστεί επίσης, εάν
άλλαξε η αντιστοίχιση του γράμματος της μονάδας δίσκου εκκίνησης. 
Αυτό μπορεί να οφείλεται σε διάφορους λόγους:
* Αντιστοιχίσατε δίσκο δικτύου στο C. Εάν έγινε 
  κάτι τέτοιο, αντιστοιχίστε ξανά το δίσκο δικτύου.
* Έχετε μια δήλωση LastDrive=C: στο αρχείο Config.sys. 
  Καταργήστε τη δήλωση LastDrive ή ρυθμίστε την, ώστε να αντιστοιχεί 
  σε άλλη μονάδα.
* Η μονάδα άλλαξε αντιστοίχιση λόγω προγράμματος διαμερισμού ή διαχείρισης
  εκκίνησης άλλου κατασκευαστή. Συμβουλευτείτε την τεκμηρίωση του προγράμματος. 
  Μπορεί να χρειαστεί να καταργήσετε την εγκατάσταση του προγράμματος πριν να εγκαταστήσετε 
  τα Windows 98.

Μήνυμα: "Ανεπανόρθωτο σφάλμα εγκατάστασης" 
"Ανεπανόρθωτο σφάλμα εγκατάστασης. Η Εγκατάσταση δεν είναι σε θέση να συνεχίσει σε αυτήν 
τη σύνθεση του συστήματος. Κάντε κλικ στο "OΚ" για να κλείσει η Εγκατάσταση." 
Αυτό το σφάλμα μπορεί να οφείλεται σε διάφορες συνθήκες. Ανατρέξτε στην ενότητα
"Γενικές σημειώσεις εγκατάστασης" και την ενότητα "ΕΓΚΑΤΑΣΤΑΣΗ ΤΩΝ WINDOWS 98 ΑΠΟ ΤΟ MS-DOS" 
για περισσότερες πληροφορίες.

Μηνύματα σφάλματος Long File Names
Εάν εμφανιστεί το μήνυμα "Η Εγκατάσταση εντόπισε την 
εφαρμογή "Long File Names" εγκατεστημένη σε αυτόν τον κατάλογο και 
δεν μπορεί να συνεχίσει.", κλείστε την Εγκατάσταση και
καταργήστε το Long File Names από τον υπολογιστή σας,
χρησιμοποιώντας τη δυνατότητα της "Κατάργησης εγκατάστασης" στο Long File
Names. Ανατρέξτε στην ενότητα View Software για περισσότερες πληροφορίες.

Μηνύματα για ανεπαρκή μνήμη
Εάν εμφανιστεί ένα μήνυμα "Η μνήμη εξαντλήθηκε", μπορείτε 
να αυξήσετε τη συμβατική μνήμη απενεργοποιώντας προγράμματα TSR 
και φορτώνοντας προγράμματα οδήγησης συσκευής στην περιοχή της ανώτερης μνήμης. 
Για πληροφορίες σχετικά με το πώς διεξαγάγετε αυτήν τη διαδικασία, ανατρέξτε 
στο ’ρθρο Q134399 της Microsoft Knowledge Base, με τίτλο How to 
Increase Conventional Memory for MS-DOS-Based Programs, στην τοποθεσία 
http://support.microsoft.com/support/kb/articles/q134/3/99.asp

Μηνύματα για ανεπάρκεια χώρου στο δίσκο
Μπορείτε να ανακτήσετε χώρο στο δίσκο κάνοντας ένα ή όλα 
τα ακόλουθα βήματα:

* Κάντε κλικ με το δεξιό κουμπί του ποντικιού στο 
  εικονίδιο "Κάδος ανακύκλωσης" και στη συνέχεια 
  κάντε κλικ στην εντολή "’δειασμα κάδου ανακύκλωσης".

* Διαγράψτε τα περιεχόμενα του φακέλου προσωρινής αποθήκευσης 
  του προγράμματος ανάγνωσης ιστοσελίδων Internet.

* Διαγράψτε τα αρχεία με επεκτάσεις .bak και .tmp.

* Διαγράψτε τους φακέλους προγραμμάτων που δεν χρησιμοποιείτε 
  (βεβαιωθείτε ότι έχετε δημιουργήσει αντίγραφα ασφαλείας
  των δεδομένων).

* Διαγράψτε τον παλιό φάκελο του MS-DOS, εκτός εάν σκοπεύετε να 
  ρυθμίσετε τις παραμέτρους του υπολογιστή σας για να λειτουργεί και με Windows 98 και 
  με MS-DOS. (Πρώτα, βεβαιωθείτε ότι έχετε δισκέτα εκκίνησης που υποστηρίζει 
  την πρόσβαση στη μονάδα δίσκου του CD-ROM.)

* Διαγράψτε το κρυφό αρχείο Winundo.dat από την προηγούμενη 
  εγκατάσταση των Windows 95.

* Διαγράψτε τον παλιό φάκελο των Windows 3.1, εκτός εάν σκοπεύετε 
  να ρυθμίσετε τις παραμέτρους του υπολογιστή σας για να λειτουργεί και με Windows 3.1 
  και με Windows 98.

"Η Εγκατάσταση δεν είναι δυνατό να γράψει στον προσωρινό κατάλογο"
Αυτό το μήνυμα μπορεί να εμφανιστεί, επειδή δεν υπάρχει αρκετός 
χώρος στο δίσκο για τον προσωρινό κατάλογο. Εάν υπάρχει διαθέσιμος 
χώρος σε άλλο δίσκο, χρησιμοποιήστε την ακόλουθη γραμμή εντολής 
για να αλλάξετε τη θέση του προσωρινού καταλόγου:

	Setup /T:<γράμμα μονάδας>:\TEMP

Εάν δεν υπάρχει διαθέσιμος χώρος σε άλλη μονάδα δίσκου, 
ελευθερώστε χώρο στο δίσκο και στη συνέχεια εκτελέστε 
ξανά την Εγκατάσταση. Ανατρέξτε στην ενότητα "Μηνύματα 
για ανεπάρκεια χώρου στο δίσκο", για να δείτε τα αρχεία 
που μπορείτε να διαγράψετε.

Μηνύματα σφάλματος αρχείου εγκατάστασης (.CAB)
Τα σφάλματα στο αρχείο εγκατάστασης (.CAB) οφείλονται συνήθως
σε έναν ή περισσότερους από τους εξής λόγους: ιοί, χαμηλή συμβατική μνήμη, 
εσφαλμένο αποθηκευτικό μέσο ή υλικό, ειδικότερα μονάδες CD-ROM. 

Εάν έχετε ήδη δημιουργήσει μια δισκέτα εκκίνησης, κλείστε την Εγκατάσταση, 
τερματίστε τη λειτουργία του υπολογιστή σας, τοποθετήστε τη δισκέτα εκκίνησης 
στη μονάδα δίσκου και κάντε επανεκκίνηση του υπολογιστή. Αυτή η ενέργεια δημιουργεί 
ένα ελεγχόμενο περιβάλλον από το οποίο μπορείτε να εκτελέσετε την Εγκατάσταση.
Στη συνέχεια, εκτελέστε την Εγκατάσταση από τη
γραμμή εντολών του MS-DOS.

Για να αποκαταστήσετε μηνύματα σφάλματος για κατεστραμμένο
αρχείο εγκατάστασης (.CAB), προσπαθήστε να αντιγράψετε 
τα αρχεία Εγκατάστασης στον σκληρό δίσκο και να εκτελέσετε 
την Εγκατάσταση από τον σκληρό δίσκο, με τα ακόλουθα βήματα:

1. Χρειάζεστε επιπλέον 200 MB ελεύθερο χώρο στον σκληρό δίσκο 
   και έναν κατάλογο temp για να αποθηκεύσετε αρχεία.
2. Κάντε εκκίνηση με τη δισκέτα εκκίνησης των Windows 98 και ενεργοποιήστε 
   την επιλογή φόρτωσης των προγραμμάτων οδήγησης του CD-ROM. Εάν 
   δεν έχετε πρόσβαση στη μονάδα του CD-ROM όταν 
   κάνετε εκκίνηση με τη δισκέτα εκκίνησης των Windows 98, 
   πρέπει να φορτώσετε τα προγράμματα οδήγησης που συνοδεύουν τη μονάδα 
   του CD-ROM.
3. Μεταβείτε στη μονάδα του CD-ROM πληκτρολογώντας το γράμμα της μονάδας δίσκου 
   που εμφανίζεται στη γραμμή εντολών MSCD001 στην οθόνη και μεταβείτε στο 
   φάκελο Win98. Πληκτρολογήστε: Copy *.* <μονάδα προορισμού 
   και το φάκελο temp που δημιουργήσατε>
4. Μεταβείτε σε αυτόν το φάκελο και εκτελέστε το πρόγραμμα Εγκατάστασης, με την επιλογή
   "Ασφαλής αποκατάσταση", εάν σας ζητηθεί.

Ανατρέξτε στην ενότητα "ΕΓΚΑΤΑΣΤΑΣΗ ΤΩΝ WINDOWS 98 ΑΠΟ ΤΟ MS-DOS"
για περισσότερες πληροφορίες.

Εάν έχετε Multimedia Cloaking και κάνετε εγκατάσταση 
των Windows 98 από δισκέτες, το πρόγραμμα Εγκατάστασης μπορεί 
να μην εκτελεστεί με επιτυχία. Εάν εμφανιστούν μηνύματα που δηλώνουν ότι η Εγκατάσταση δεν
μπορεί να διαβάσει τα αρχεία .cab, ακολουθήστε την εξής διαδικασία:

1. Καταργήστε τη γραμμή όπου αναφέρεται το αρχείο Cacheclk.exe από
   τα αρχεία Config.sys και Autoexec.bat.
2. Ξεκινήστε πάλι τον υπολογιστή σας.
3. Εκτελέστε ξανά το πρόγραμμα Εγκατάστασης.

Μήνυμα SU0010, SU0012, SU0015 ή SU0016
Εάν εμφανιστεί ένα από αυτά τα μηνύματα κατά την Εγκατάσταση, 
ανατρέξτε στις ενότητες "Εγκατάσταση των Windows 98 σε σύστημα που 
χρησιμοποιεί Windows NT" και "Εγκατάσταση των Windows 98 σε σύστημα

όπου c:\windows είναι η διαδρομή προς τον κατάλογο των Windows.

"Η Εγκατάσταση δεν έχει αρκετή διαθέσιμη συμβατική μνήμη για να ελέγξει

"Η Εγκατάσταση δεν ήταν δυνατό να ελέγξει τους σκληρούς δίσκους του
υπολογιστή σας".

Εάν εμφανιστεί ένα από αυτά τα δύο μηνύματα σφάλματος, η Εγκατάσταση δεν ήταν σε θέση να 
εκτελέσει την Εξέταση Δίσκων για να ελέγξει τους σκληρούς δίσκους. 
Αυτό μπορεί να συμβαίνει λόγω ανεπαρκούς συμβατικής μνήμης ή επειδή ο δίσκος 
περιέχει σφάλματα που δεν μπορεί να διορθώσει η Εξέταση Δίσκων, όταν

τα σφάλματα, ξεκινήστε το σύστημα σε λειτουργία

αποτύχει πάλι, κάντε επανεκκίνηση σε λειτουργία MS-DOS 
και εκτελέστε την εντολή Scandisk /all /surface. Αυτό θα διαρκέσει 
λίγο περισσότερο, αλλά θα βεβαιωθείτε ότι οι σκληροί δίσκοι 
βρίσκονται σε καλή κατάσταση. Εάν, μετά την εκτέλεση της
Εξέτασης Δίσκων με αυτές τις επιλογές, η Εγκατάσταση δεν συνεχίζει, 
θα πρέπει να εκτελέσετε την εντολή Setup /is για να παρακάμψετε

ΣΗΜΕΙΩΣΗ: Δεν προτείνεται η παράκαμψη της Εξέτασης Δίσκων και θα πρέπει
να χρησιμοποιείται μόνο εάν είστε σίγουροι ότι ο(οι) δίσκος(οι) δεν
περιέχει(ουν) σφάλματα. Εάν υπάρχουν ακόμη σφάλματα,

να κάνετε εκκίνηση του συστήματος.

ΕΠΕΞΕΡΓΑΣΙΑ ΤΩΝ ΑΡΧΕΙΩΝ CONFIG.SYS ΚΑΙ AUTOEXEC.BAT
Τα αρχεία Config.sys και Autoexec.bat του υπολογιστή λένε 
στον υπολογιστή ποια προγράμματα και ποιες συσκευές να φορτώσει 
κατά την εκκίνηση (για παράδειγμα, ένα πρόγραμμα ανίχνευσης ιών 
στο Autoexec.bat μπορεί να καθοδηγεί τον υπολογιστή να το φορτώνει 
αυτόματα). Η Εγκατάσταση των Windows 98 μπορεί να μην λειτουργεί
σωστά με ορισμένα προγράμματα και ορισμένες συσκευές. 
Για να καταργήσετε ή να απενεργοποιήσετε ένα τέτοιο πρόγραμμα ή συσκευή,
μπορεί να χρειαστεί να επεξεργαστείτε τα αρχεία Config.sys 
ή/και Autoexec.bat.

Για να επεξεργαστείτε τα αρχεία Config.sys και Autoexec.bat:

1. Στα Windows 3.1 ή 3.11, κάντε κλικ στο μενού "Αρχείο", κάντε κλικ στην εντολή "Εκτέλεση", 
   πληκτρολογήστε Sysedit και στη συνέχεια πιέστε το πλήκτρο ENTER. Στα Windows 95, 
   κάντε κλικ στο κουμπί "Έναρξη", κάντε κλικ στην εντολή "Εκτέλεση", πληκτρολογήστε
   Sysedit και στη συνέχεια πιέστε το πλήκτρο ENTER.

2. Στο παράθυρο διαλόγου Config.sys ή Autoexec.bat, πληκτρολογήστε 
   REM στην αρχή οποιασδήποτε γραμμής(ών) θέλετε 
   να απενεργοποιήσετε.

3. Αποθηκεύστε τις αλλαγές και ξεκινήστε πάλι τον υπολογιστή.

Κάρτες δικτύου Τοποθέτησης και ’μεσης Λειτουργίας και Προγράμματα

ΡΥΘΜΙΣΗ ΠΑΡΑΜΕΤΡΩΝ ΔΙΠΛΗΣ ΕΚΚΙΝΗΣΗΣ ΜΕ ΤΑ WINDOWS NT 
Για να ρυθμίσετε παραμέτρους διπλής εκκίνησης σε έναν
υπολογιστή x86, εγκαταστήστε το λειτουργικό σύστημα
κανονικά και στη συνέχεια επεξεργαστείτε το αρχείο
Boot.ini, όπως περιγράφεται παρακάτω. Όλες οι πληροφορίες 
εκκίνησης συστήματος αποθηκεύονται στο αρχείο Boot.ini, 
το οποίο δημιουργείται αυτόματα κατά την Εγκατάσταση 
στη ρίζα του σκληρού δίσκου του υπολογιστή σας.

>>>Επεξεργασία του αρχείου Boot.ini:

1. Στην Εξερεύνηση των Windows, κάντε κλικ στο μενού "Προβολή", κάντε κλικ 
   στην εντολή "Επιλογές" και στη συνέχεια ενεργοποιήστε την επιλογή
   "Εμφάνιση όλων των αρχείων".

2. Βεβαιωθείτε ότι δεν είναι ενεργοποιημένη η επιλογή 
   "Απόκρυψη επεκτάσεων αρχείου για γνωστούς τύπους αρχείων", 
   και στη συνέχεια κάντε κλικ στο κουμπί "OK".

3. Κάντε κλικ με το δεξιό κουμπί του ποντικιού στο αρχείο Boot.ini 
   και στη συνέχεια κάντε κλικ στην εντολή "Ιδιότητες".

4. Κάντε κλικ για να καταργήσετε την επιλογή από το πλαίσιο ελέγχου "Μόνο για ανάγνωση", 
   και στη συνέχεια κάντε κλικ στο κουμπί "OK".

5. Κάντε κλικ με το δεξιό κουμπί του ποντικιού στο αρχείο
   Boot.ini, κάντε κλικ στην εντολή "Αντιγραφή", κάντε κλικ 
   με το δεξιό κουμπί του ποντικιού σε ένα κενό χώρο του παραθύρου
   διαλόγου της Εξερεύνησης και στη συνέχεια κάντε κλικ στην 
   εντολή "Επικόλληση". Θα δημιουργηθεί ένα αντίγραφο ασφαλείας 
   με το όνομα αρχείου "Αντίγραφο του Boot.ini".

6. Κάντε διπλό κλικ στο αρχείο Boot.ini.

7. Προσθέστε το όνομα και τη θέση του εναλλακτικού συστήματος στην 
   ενότητα [operating systems] του αρχείου, όπως στο ακόλουθο 
   παράδειγμα:

	[operating systems]
	C:\Winnt="Windows NT 4.0"
	C:\="Microsoft Windows"

8. Αποθηκεύστε και κλείστε το αρχείο Boot.ini.

9. Κάντε κλικ με το δεξιό κουμπί του ποντικιού στο αρχείο Boot.ini 
   και στη συνέχεια κάντε κλικ στην εντολή "Ιδιότητες".

10.Επιλέξτε το πλαίσιο ελέγχου "Μόνο για ανάγνωση" 
   και στη συνέχεια κάντε κλικ στο κουμπί "ΟΚ".

ΔΗΜΙΟΥΡΓΙΑ ΔΙΑΜΕΡΙΣΜΑΤΟΣ ΕΚΚΙΝΗΣΗΣ ΤΟΥ MS-DOS
Για να εγκαταστήσετε τα Windows 98, η μονάδα δίσκου εκκίνησης 
πρέπει να είναι διαμέρισμα εκκίνησης του MS-DOS. Εάν η μονάδα δίσκου
εκκίνησης είναι διαμορφωμένη ως HPFS ή NTFS, πρέπει να δημιουργήσετε
ένα διαμέρισμα εκκίνησης του MS-DOS πριν να εκτελέσετε την Εγκατάσταση. 
Για περισσότερες πληροφορίες σχετικά με τη δημιουργία διαμερίσματος
εκκίνησης του MS-DOS, δείτε την τεκμηρίωση του υπολογιστή σας.

ΧΡΗΣΗ ΠΡΟΓΡΑΜΜΑΤΩΝ ΚΑΙ ΠΡΟΓΡΑΜΜΑΤΩΝ
ΟΔΗΓΗΣΗΣ TSR 
APPEND (MS-DOS)
Μην χρησιμοποιείτε το βοηθητικό πρόγραμμα APPEND 
με τα Windows 98. Εμποδίζει τα Windows και τα προγράμματα
των Windows να δημιουργήσουν έγκυρα ονόματα 
Well-Formed-Path (WFP) για τα αρχεία που χρησιμοποιούν. 
(Το WFP είναι το πλήρες όνομα ενός αρχείου, συμπεριλαμβανομένου
του γράμματος της μονάδας δίσκου και της πλήρους διαδρομής, 
ξεκινώντας από τον ριζικό κατάλογο.)

Autocon Έκδοση 2.0e, Bootcon Έκδοση 1.60 και Boot.sys
Αυτά τα προγράμματα σας επιτρέπουν να επιλέξετε διάφορα 
σύνολα παραμέτρων, όταν ξεκινάτε τον υπολογιστή σας. 
Επειδή η Εγκατάσταση των Windows 98 τροποποιεί μόνο
τις πρώτες ενότητες των αρχείων Config.sys και Autoexec.bat, 
ίσως χρειαστεί να τροποποιήσετε αυτά τα αρχεία εσείς
μετά την Εγκατάσταση. Για περισσότερες πληροφορίες, 
ανατρέξτε στην τεκμηρίωση του προγράμματος.

Cubit Έκδοση 3.01
Όταν χρησιμοποιείτε το Cubit με τα Windows 98, μην 
συμπιέζετε το φάκελο System των Windows ή οποιοδήποτε
αρχείο που φορτώνεται από το αρχείο Config.sys.

Πριν να κάνετε αναβάθμιση στα Windows 98, βεβαιωθείτε 
ότι η παλαιότερη έκδοση των Windows έχει αποσυμπιεστεί 
στον σκληρό σας δίσκο. Μπορείτε να ξανασυμπιέσετε τα
αρχεία, αφού εγκαταστήσετε τα Windows 98 με επιτυχία.

Doubledisk Έκδοση 2.5
Τα Windows 98 μπορεί να προσπαθήσουν να αποκτήσουν
πρόσβαση σε ανύπαρκτες μονάδες δίσκου, εάν τα
χρησιμοποιήσετε με το Doubledisk. Οι ανύπαρκτες μονάδες δίσκου 
προκύπτουν, όταν χρησιμοποιείτε το Doubledisk για να
δημιουργήσετε μια συμπιεσμένη μονάδα δίσκου. 
Για παράδειγμα, αν ο υπολογιστής σας έχει τις μονάδες
δίσκου A, B και C και εσείς χρησιμοποιήσετε το Doubledisk 
για να δημιουργήσετε τη μονάδα δίσκου F, τα Windows 98 
και το MS-DOS αναγνωρίζουν τις μονάδες δίσκου D και E 
ως έγκυρες μονάδες δίσκου, ακόμη και αν δεν υπάρχουν. 
Μπορείτε να χρησιμοποιήσετε το βοηθητικό πρόγραμμα 
DRVOFF, για να εμποδίσετε τα Windows να αναγνωρίσουν 
ανύπαρκτες μονάδες. Για να παραγγείλετε ένα αντίγραφο
του βοηθητικού προγράμματος DRVOFF, επικοινωνήστε
με την Vertisoft.

Infinite Disk Έκδοση 2.1
Το Infinite Disk έκδοση 2.1 δεν λειτουργεί σωστά με τα 
Windows 98. Καταργήστε το λογισμικό Infinite Disk 
από τον υπολογιστή σας πριν να εκτελέσετε το πρόγραμμα
Εγκατάστασης ή επικοινωνήστε με την εταιρεία εμπορίας λογισμικού
που μπορεί να σας προμηθεύσει μια έκδοση του λογισμικού
Infinite Disk συμβατή με τα Windows 98.

>>>Κατάργηση του Infinite Disk από τον υπολογιστή σας:

1. Αποσυμπιέστε οποιαδήποτε συμπιεσμένα αρχεία.

2. Καταργήστε τις ακόλουθες γραμμές από τα αρχεία Config.sys 
   και Autoexec.bat:

	iddrv.sys
	idres.exe
	protect /c

3. Ξεκινήστε πάλι τον υπολογιστή σας και στη συνέχεια εκτελέστε
   ξανά το πρόγραμμα Εγκατάστασης.

NetWare(R) NetBIOS
Σε πολλές ρυθμίσεις παραμέτρων, το NetWare NetBIOS TSR 
δεν είναι συμβατό με τα Windows 98. Στις περισσότερες περιπτώσεις, 
η Εγκατάσταση θα εντοπίσει το NetWare NetBIOS και θα
καταργήσει την εντολή εκκίνησης από τον υπολογιστή σας. 
Ωστόσο, εάν έχετε εγκαταστήσει το NetWare NetBIOS, 
ώστε να κάνει εκκίνηση από άλλη θέση και όχι από το αρχείο 
Autoexec.bat (για παράδειγμα, από διαφορετικό αρχείο δέσμης), 
η Εγκατάσταση δεν μπορεί να το εντοπίσει. Σε αυτήν την περίπτωση, 
πρέπει να καταργήσετε το NetWare NetBIOS εσείς. 
Τα Windows 98 περιλαμβάνουν ένα πρωτόκολλο που είναι συμβατό
με το IPX/SPX και το οποίο μπορείτε να χρησιμοποιήσετε αντί του 
NetBIOS. Για πληροφορίες σχετικά με τη ρύθμιση των παραμέτρων
των Windows 98 ώστε να λειτουργούν με αυτό, ανατρέξτε
στο αρχείο Networks.txt.

Newspace Έκδοση 1.07
Τα Windows 98 δεν είναι συμβατά με αυτό το βοηθητικό
πρόγραμμα συμπίεσης δίσκων.

Norton Desktop για Windows (Ep.exe)
Εάν το πρόγραμμα ανάκτησης δεδομένων Erase Protect (Ep.exe) 
έχει φορτωθεί από το αρχείο Autoexec.bat, απενεργοποιήστε το 
όταν πρόκειται να εγκαταστήσετε τα Windows, ιδιαίτερα εάν
κάνετε αναβάθμιση από προηγούμενη έκδοση των Windows. 
Επειδή το Erase Protect προσπαθεί να αποθηκεύσει αντίγραφα 
όλων των διαγραμμένων αρχείων, μπορεί να εξαντληθεί ο χώρος
του δίσκου κατά την Εγκατάσταση.

Προτείνεται να εκτελείτε το Erase Protect με ένα μόνιμο 
αρχείο μετάθεσης. Εάν χρησιμοποιείτε ένα προσωρινό αρχείο 
μετάθεσης, εξαιρέστε τα αρχεία με επέκταση .tmp ή .swp 
από την προστασία διαγραφής.

Norton Disklock Εκδόσεις 3.01 και 3.5
Οι εκδόσεις 3.01 και 3.5 του Norton Disklock δεν λειτουργούν 
σωστά με τα Windows 98. Καταργήστε αυτό το λογισμικό 
από τον υπολογιστή σας πριν να εκτελέσετε το πρόγραμμα 
Εγκατάστασης ή επικοινωνήστε με την υπηρεσία υποστήριξης για προϊόντα της Symantec, 
για να λάβετε έκδοση του Disklock που είναι συμβατή με τα Windows 98.

Για πληροφορίες σχετικά με τον τρόπο κατάργησης του Disklock 
από τον υπολογιστή σας, ανατρέξτε στην τεκμηρίωση του 
Norton Disklock.

PC Tools Deluxe(TM) Έκδοση 7.1 (DATAMON)
Εάν φορτώσετε τα PC Tools DATAMON χρησιμοποιώντας
την επιλογή \SENTRY+, η Εγκατάσταση μπορεί να αποτύχει
εάν έχετε λίγο χώρο στο δίσκο. Μην χρησιμοποιείτε την επιλογή 
\SENTRY+ όταν φορτώνετε το DATAMON.

Μην φορτώνετε το DATAMON αφού έχετε φορτώσει το
SmartDrive. Εάν φορτωθεί το DATAMON μετά το SmartDrive, 
ο υπολογιστής σας μπορεί να παρουσιάσει σφάλμα. Ανατρέξτε στην 
τεκμηρίωση των PC Tools για πληροφορίες σχετικά με τον τρόπο 
κατάργησης του DATAMON.

PC Tools Deluxe Έκδοση 6.0 (DESKTOP)
Εάν προσπαθήσετε να ξεκινήσετε το PC Tools 
από τη γραμμή εντολών μέσα από τα Windows, ο υπολογιστής σας 
μπορεί να παρουσιάσει σφάλμα ή να κάνει επανεκκίνηση 
μόνος του. Επίσης, αυτά τα θέματα προκύπτουν, όταν 
χρησιμοποιείτε εντολές του MS-DOS από μια γραμμή εντολών 
μέσα από τα Windows.

Η Έκδοση 7.0 του PC Tools μπορεί να διορθώσει αυτά τα θέματα. 
Ωστόσο, μην φορτώνετε το DESKTOP από γραμμή εντολών, 
εάν είχε φορτωθεί πριν ξεκινήσετε τα Windows. Εάν το κάνετε, 
ο υπολογιστής σας μπορεί να παρουσιάσει σφάλμα κατά τον τερματισμό 
των Windows. Προτείνεται να εκτελέσετε το DESKTOP 
από το αρχείο .PIF του, που παρέχεται με τα Windows 98.

SUBST (MS-DOS)
Μπορεί να αντιμετωπίσετε δυσκολίες με το SUBST 
κατά την Εγκατάσταση, αλλά μπορείτε να το προσθέσετε 
και πάλι με ασφάλεια, αφού εγκαταστήσετε τα 
Windows 98.

UNDELETE
Εάν εκτελείτε την εντολή UNDELETE με την επιλογή SENTRY, 
η Εγκατάσταση μπορεί να εμφανίσει προβλήματα και μπορεί 
να εξαντληθεί ο ελεύθερος χώρος στο δίσκο κατά την 
Εγκατάσταση. Προτείνεται να καταργήσετε την εντολή UNDELETE 
από τα αρχεία Config.sys και Autoexec.bat πριν να εκτελέσετε 
το πρόγραμμα Εγκατάστασης.

ΣΗΜΕΙΩΣΗ: Η επιλογή SENTRY δημιουργεί έναν κρυφό 
κατάλογο που ονομάζεται Sentry. Όταν διαγράφετε ένα αρχείο, 
η εντολή UNDELETE μετακινεί το αρχείο από την τρέχουσα θέση του 
στον κατάλογο Sentry. Πριν να καταργήσετε την εντολή 
UNDELETE από το αρχείο Config.sys ή το αρχείο Autoexec.bat, 
ελευθερώστε χώρο στο δίσκο χρησιμοποιώντας την εντολή 
UNDELETE/PURGE σε όλες τις μονάδες δίσκου.

Εάν δεν καταργήσετε την εντολή Undelete, η Εγκατάσταση 
θα την απενεργοποιήσει.

ΓΝΩΣΤΑ ΠΙΘΑΝΑ ΘΕΜΑΤΑ ΕΓΚΑΤΑΣΤΑΣΗΣ
Εάν αντιμετωπίσετε σφάλματα κατά την Εγκατάσταση, 
όπως μη αναμενόμενο κλείσιμο της Εγκατάστασης ή 
μηνύματα σφάλματος που δεν ισχύουν, 
αυξήστε το μέγεθος του μόνιμου αρχείου μετάθεσης. 
Για να ολοκληρωθεί η Εγκατάσταση με επιτυχία, 
το μέγεθος του μόνιμου αρχείου μετάθεσης συν η μνήμη 
RAM του υπολογιστή σας πρέπει να είναι
τουλάχιστον 14 MB. Αφού αλλάξετε το μέγεθος του αρχείου 
μετάθεσης, εκτελέστε ξανά την Εγκατάσταση.

ΣΗΜΕΙΩΣΗ: Εάν ο σκληρός σας δίσκος είναι συμπιεσμένος, 
μπορεί να χρειαστεί να αυξήσετε το μέγεθος της μονάδας υποδοχής 
για να μπορέσετε να αυξήσετε το μέγεθος του αρχείου μετάθεσης.


ΣΗΜΕΙΩΣΗ: Για να μάθετε περισσότερα σχετικά με την εγκατάσταση
των Πανευρωπαϊκών ρυθμίσεων, ανατρέξτε στο αρχείο Intl.txt.
