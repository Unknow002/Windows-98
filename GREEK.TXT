και σχόλια για κάποια θέματα που έχουν διαπιστωθεί στην Ελληνική Έκδοση
των Windows 98.

  4. Αναβάθμιση της Αμερικάνικης έκδοσης των Windows 98
  5. Υποστήριξη για τους ελληνικούς χαρακτήρες σε δίκτυο
  6. Ελληνικά αρχεία κειμένου DOS (737, 869) και Windows 98
  7. Αρχεία .Wri των Windows 3.x
  8. Αποθήκευση αρχείων στο Wordpad 
  9. Ελληνικά προγράμματα οδήγησης εκτυπωτών ακίδος
 10. Προφίλ υλικού

1. Εγκατάσταση ρυθμίσεων για τα Ελληνικά
----------------------------------------
Κατά την εγκατάσταση των Ελληνικών Windows 98 ίσως χρειαστεί να επιλέξετε 
ελληνική κωδικοσελίδα από τις Τοπικές Ρυθμίσεις και ελληνική διάταξη 
πληκτρολογίου, όταν εμφανιστεί το παράθυρο διαλόγου Ρυθμίσεις Υπολογιστή.
Υπάρχουν δύο Ελληνικά Windows 98, που υποστηρίζουν αντίστοιχα τις δύο 
διαφορετικές κωδικοσελίδες για το MS-DOS.

 - Η έκδοση 737, που υποστηρίζει την κωδικοσελίδα 737 για το MS-DOS.
 - Η έκδοση 869, που υποστηρίζει την κωδικοσελίδα 869 για το MS-DOS.

Η προεπιλεγμένη έκδοση εγκατάστασης είναι κωδικοσελίδας 737 και 
βρίσκεται στον κατάλογο των Win98 στο CD-ROM.
Μπορείτε να εγκαταστήσετε την έκδοση κωδικοσελίδας 869 από τον κατάλογο 
Win98.869 του CD-ROM.

Δεν μπορείτε να αλλάξετε κωδικοσελίδα μετά από την εγκατάσταση!

Η Ελληνική κωδικοσελίδα 1253 για τα Windows θα επιλεγεί αυτόματα, 
οποιαδήποτε από τις κωδικοσελίδες 737 και 869 και αν επιλέξετε για το DOS.

Για ελληνική διάταξη πληκτρολογίου, μπορείτε να επιλέξετε:

α) Το Ελληνικό πληκτρολόγιο 101-πλήκτρων 
β) Το Ελληνικό πληκτρολόγιο IBM 319, που ονομάζεται και ΕΛΟΤ-1000 
γ) Το Ελληνικό πληκτρολόγιο IBM 220

Και η Ελληνική και η Αγγλική διάταξη πληκτρολογίου εγκαθίστανται για όλα τα 
παραπάνω πληκτρολόγια.

Για εναλλαγή της ελληνικής και της αγγλικής γλώσσας στο πληκτρολόγιο κατά
τη διάρκεια της εγκατάστασης πρέπει να πατήσετε : 
  Αριστερό και Δεξιό Shift
Για εναλλαγή της ελληνικής και της αγγλικής γλώσσας στο πληκτρολόγιο, κατά
τη λειτουργία των Windows πρέπει να πατήσετε :
  Αριστερό Alt+Shift

Για να προσθέσετε ή να αλλάξετε διάταξη πληκτρολογίου μετά από την 
εγκατάσταση, πρέπει να επιλέξετε:
	Έναρξη-Ρυθμίσεις-Πίνακας Ελέγχου-Πληκτρολόγιο-Γλώσσα.

Για να αλλάξετε μια από τις υπάρχουσες διατάξεις πληκτρολογίου, επιλέξτε την 
και κάντε κλικ στο Ιδιότητες.

Για να χρησιμοποιήσετε ελληνικό πληκτρολόγιο IBM 319 στο DOS, πρέπει να 
προσθέσετε την παράμετρο /id:319 στην υπάρχουσα καταχώριση πληκτρολογίου 
στο autoexec.bat. Για πληκτρολόγιο IBM 220, θα πρέπει να προσθέσετε 
την παράμετρο /id:220.


2. Αναβαθμίσεις για άλλες εκδόσεις των Windows
----------------------------------------------
Για τους χρήστες των Ελληνικών Windows 95, Paneuro Windows 95 με Ελληνικές
ρυθμίσεις, Ελληνική έκδοση των Win95 OSR2 και Αμερικάνικα Windows 98 με Ελληνικές 
ρυθμίσεις, κατά την αναβάθμιση η Εγκατάσταση θα ανιχνεύσει και θα διατηρήσει 
την κωδικοσελίδα και τις ρυθμίσεις πληκτρολογίου που έχετε.
Π.χ. Αν θέλετε να εγκαταστήσετε την έκδοση 737 για το ΜS-DOS εκτελέστε 
την Εγκατάσταση από τον κατάλογο Win98. Εάν θέλετε να εγκαταστήσετε την 
έκδοση 869 για το ΜS-DOS εκτελέστε την Εγκατάσταση από τον κατάλογο Win98.869.
Οι χρήστες της έκδοσης 851 για το ΜS-DOS πρέπει να εγκαταστήσουν την έκδοση 869.
ΠΡΟΣΟΧΗ: Εάν εγκαταστήσετε εσφαλμένη κωδικοσελίδα για το ΜS-DOS, είναι πιθανό
οι εφαρμογές σας για το ΜS-DOS να μη λειτουργούν σωστά και ονόματα αρχείων να 
καταστραφούν.


3. Αναβαθμίσεις για Windows 3.1 και Windows for Workgroups
----------------------------------------------------------
Προτείνεται να αναβαθμίσετε μια υπάρχουσα εγκατάσταση των Windows, 
εκτελώντας το πρόγραμμα εγκατάστασης από το MS DOS. Για συστήματα 
με κωδικοσελίδα 737 εκτελέστε την εγκατάσταση από τον κατάλογο Win98,
ενώ για συστήματα με κωδικοσελίδα 869 εκτελέστε την εγκατάσταση 
από τον κατάλογο Win98.869. Για περισσότερες λεπτομέρειες σχετικά με 
τη διαδικασία αναβάθμισης μπορείτε να ανατρέξετε στο φυλλάδιο 
"Οδηγίες Εγκατάστασης" που περιλαμβάνεται στο πακέτο του λογισμικού.

α) Κατά την αναβάθμιση, οι υπάρχουσες ελληνικές ρυθμίσεις θα ανιχνευθούν 
   και θα διατηρηθούν!

β) Κατά την αναβάθμιση, οι παρακάτω παλιές εφαρμογές των Windows 3.x
   θα διαγραφούν:
     calendar.exe
     recorder.exe
     clock.exe
     terminal.exe
     soundrec.exe
     sol.exe
     csconvert.exe

   Το βοήθημα Csconvert.exe για τα Ελληνικά Windows 95 θα είναι διαθέσιμο. 



4. Αναβάθμιση της Αμερικάνικης έκδοσης των Windows 98
-----------------------------------------------------
Μετά από την αναβάθμιση της Αμερικάνικης έκδοσης των Windows 98, 
δημιουργείται στην επιφάνεια εργασίας ένα εικονίδιο με το όνομα Old Desktop 
(παλιά επιφάνεια εργασίας). Μέσα σ' αυτό βρίσκονται ο προηγούμενος 
χαρτοφύλακας (briefcase), καθώς και τα έγγραφα που περιέχει, και  το
εικονίδιο που αναφέρεται στην εγκατάσταση του Microsoft Network, (Setup 
The Microsoft Network), αν δεν εγκαταστήσατε το MSN. Μπορείτε ελεύθερα να 
διαγράψετε το εικονίδιο αυτό σύροντάς το στον Κάδο ανακύκλωσης.
Αν είχατε εγκαταστήσει το MSN θα έχετε δύο εικονίδια, το "ΤΟ Microsoft  
Network" και το "Εγκατάσταση του Microsoft Network". Αφού έχετε ήδη 
εγκαταστήσει το MSN μπορείτε ελεύθερα να διαγράψετε το δεύτερο εικονίδιο
σύροντάς το στον Κάδο ανακύκλωσης.

Αν προσθέσατε κάποια προγράμματα στις ομάδες Accessories ή Startup και κάνετε
αναβάθμιση, οι ομάδες αυτές θα διατηρηθούν. Για να μετακινήσετε τα 
προγράμματα αυτά στις ομάδες Βοηθήματα ή Εκκίνηση πηγαίνετε στο:
  Έναρξη-Ρυθμίσεις-Γραμμή Εργασιών-Προγράμματα και επιλέξτε
  "Για προχωρημένους"
Θα έχετε:
 Start Menu
    Προγράμματα 
        Accessories
        Startup
        Βοηθήματα
        Εκκίνηση  

Διάκενο εικονιδίων
------------------
Τα Ελληνικά Windows 98 παίρνουν τις ρυθμίσεις της Αμερικάνικης έκδοσης 
κατά την αναβάθμιση. Αυτό έχει ως αποτέλεσμα να περικοπούν μερικά ονόματα 
εικονιδίων στην επιφάνεια εργασίας και στον Πίνακα Ελέγχου. Για να αλλάξετε 
την απόσταση των εικονιδίων, πηγαίνετε στο:
  Έναρξη-Ρυθμίσεις-Πίνακας Ελέγχου-Οθόνη-Εμφάνιση και επιλέξτε 
 "Διάκενο εικονιδίων (οριζόντιο)" από το πλαίσιο λίστας "Αντικείμενο". 
Έπειτα, αυξήστε το μέγεθος από 43 σε 65. 



5. Υποστήριξη για τους ελληνικούς χαρακτήρες σε δίκτυο
------------------------------------------------------
Οι Ελληνικοί χαρακτήρες υποστηρίζονται στα ονόματα υπολογιστών και στα 
ονόματα κοινόχρηστων στοιχείων.

α) Ο Primary Domain Controller των NT πρέπει να διαθέτει υποστήριξη 
   ελληνικών προκειμένου να μπορεί να επικυρώνει συστήματα με Ελληνικά
   Windows 95 και Ελληνικά ονόματα υπολογιστή.
β) Όλα τα συστήματα (NT και Windows 98) πρέπει να χρησιμοποιούν την ίδια 
   κωδικοσελίδα DOS (737 or 869) στο όνομα περιοχής όσο και στο όνομα
   ομάδας εργασίας.
γ) Τα υπάρχοντα Ελληνικά Windows 3.1, Αμερικανικά Windows 3.1 ή Windows for 
   Workgroups με Ελληνική υποστήριξη της Microsoft ή Πουλιάδη δεν θα
   μπορούν να έχουν πρόσβαση σε πόρους σε συστήματα με Ελληνικά Windows 98 
   με Ελληνικά ονόματα υπολογιστών ή κοινόχρηστων στοιχείων.
δ) Τα συστήματα με Ελληνικά Windows 98 που ξεκινούν τη λειτουργία τους σε 
   λειτουργία MS-DOS πρέπει να χρησιμοποιούν κεφαλαίους Ελληνικούς 
   χαρακτήρες για σύνδεση με συστήματα με Ελληνικά ονόματα υπολογιστών 
   ή κοινόχρηστων στοιχείων. Για πεζούς τονούμενους χαρακτήρες, πρέπει να
   χρησιμοποιούνται οι αντίστοιχοι τονούμενοι κεφαλαίοι χαρακτήρες.


6. Ελληνικά αρχεία κειμένου DOS (737, 869) και Windows 98
---------------------------------------------------------
Για να ανοίξετε ένα αρχείο κειμένου DOS στα Windows 98, χρησιμοποιήστε το
Wordpad και ανοίξτε το αρχείο ως αρχείο με μορφή 
"Αρχεία κειμένου - Μορφή MS-DOS (*.txt)".
Το Σημειωματάριο θα ανοίξει το αρχείο ως αρχείο κειμένου Windows 
(Κωδικοσελίδα 1253) και όχι ως αρχείο κειμένου DOS (Κωδικοσελίδα 737 ή 869). 

Για να αποκόψετε και να αντιγράψετε Αγγλικούς χαρακτήρες από τα Windows στο 
edit του DOS πρέπει να είναι ενεργό το Αγγλικό πληκτρολόγιο. 


7. Αρχεία .Wri των Windows 3.x
------------------------------
Για να ανοίξετε αρχεία με επέκταση .wri που δημιουργήθηκαν με 
την εφαρμογή Write.exe των Windows 3.x, χρησιμοποιήστε την εφαρμογή
Wordpad.exe, που βρίσκεται στον κατάλογο Tools\oldwin95\write και
μετατρέψτε τα αρχεία.


8. Αποθήκευση αρχείων στο Wordpad 
---------------------------------
Το Wordpad μετατρέπει τους χαρακτήρες της κωδικοσελίδας 1253 σε χαρακτήρες 
της κωδικοσελίδας του DOS στα πλαίσια διαλόγου "’νοιγμα" και "Αποθήκευση ως".
Ως αποτέλεσμα, σε ένα σύστημα με κωδικοσελίδα DOS 737, το Wordpad δεν θα 
μπορεί να ανοίξει ή να αποθηκεύσει ένα αρχείο σε φάκελο με τους Ελληνικούς
χαρακτήρες γιώτα ή ύψιλον με τόνο και διαλυτικά. Αυτοί οι χαρακτήρες δεν 
υπάρχουν στην κωδικοσελίδα 737.

10. Προφίλ υλικού
-----------------
Σε ένα σύστημα με Windows 98 μπορούν να φορτωθούν πολλά προφίλ υλικού 
μέσα από την καρτέλα "Προφίλ υλικού" στη βοηθητική εφαρμογή Σύστημα του 
Πίνακα Ελέγχου. Για την ονομασία αυτών των προφίλ χρησιμοποιήστε μόνο 
Λατινικούς χαρακτήρες. Θα σας ζητηθεί να επιλέξετε ένα όνομα κατά την 
εκκίνηση του συστήματος σε πρόωρο στάδιο, πριν φορτωθούν οι Ελληνικές 
γραμματοσειρές.

Κατά την ώρα της εκκίνησης εμφανίζεται το μήνυμα:  
    
   Windows cannot determine what configuration your computer is in.
   Select one of the following:

   1. Original Configuration
   2. <Προφίλ υλικού με λατινικούς χαρακτήρες>
   3. None of the above
   
   Enter your choice:"


11. Προσθήκη καταχωρίσεων στο autoexec.bat
------------------------------------------
Όταν προσθέτετε νέες καταχωρίσεις στο autoexec.bat, βεβαιωθείτε ότι τις
τοποθετείτε μετά την εντολή mode που φορτώνει τις Ελληνικές γραμματοσειρές.
Σε αντίθετη περίπτωση, μπορεί να εμφανιστεί κάποιο Ελληνικό μήνυμα λάθος    
στην οθόνη.


12. Υποστήριξη ελληνικών ονομάτων αρχείων στο Netware
-----------------------------------------------------
Για υποστήριξη Ελληνικών ονομάτων αρχείων με το Netware 3.12 πρέπει να 
καταργήσετε το αρχείο lconfig.sys από τον κατάλογο ρίζας του διακομιστή
(c:\server312).


13. Για χρήστες του Microsoft Batch 98
--------------------------------------
Εάν χρησιμοποιείτε το Microsoft Batch 98 για αναβάθμιση από Microsoft Windows 95
χρησιμοποιείστε την παρακάτω εντολή για αυτόματη εγκατάσταση : 
	setupcor c:\msbatch.inf
