ΛΙΣΤΑ ΑΡΧΕΙΩΝ README ΤΩΝ WINDOWS 98
===================================

Επιπλέον του αρχείου Readme.txt, τα Windows 98 παρέχουν
τα παρακάτω αρχεία readme στον κατάλογο C:\Windows
	

Config.txt      Περιέχει πληροφορίες σύνταξης εντολών 
		του αρχείου Cοnfig.sys.

Display.txt     Παρέχει πληροφορίες για το πως να ρυθμίσετε
                παραμέτρους και να διορθώσετε σφάλματα των 
                διαθέσιμων προγραμμάτων οδήγησης και για το πως 
                να λάβετε πρόσθετα προγράμματα οδήγησης οθόνης.

Faq.txt         Απαντά στις πιο συνήθεις ερωτήσεις
		για τα Windows 98.

General.txt     Παρέχει πληροφορίες για τα προβλήματα εκκίνησης,
		για τα προγράμματα που παίρνετε με τα Windows 98, 
                για τα εργαλεία δίσκων, για τους δίσκους και τα CD, 
                για τα προγράμματα οδήγησης και αφαιρούμενα μέσα αποθήκευσης.

		Αυτό το αρχείο επίσης περιέχει τις τελευταίες πληροφορίες
		που δεν συμπεριλαμβάνονται στα άλλα αρχεία readme. 
		Για παράδειγμα, αν έχετε κάποια ερώτηση σχετικά με κάποιον 
		εκτυπωτή, είναι χρήσιμο να κοιτάξετε στο αρχείο General.txt
		καθώς και στο αρχείο Printers.txt.
	
Hardware.txt    Παρέχει πληροφορίες για γνωστά προβλήματα
                και διορθώσεις συσκευών υλικού. 
		Είναι πιθανό να χρειάζεται να ανατρέξετε στο 
		αρχείο Printers.txt ή Mouse.txt για ειδικά προβλήματα.		   

Mouse.txt       Παρέχει πληροφορίες για γνωστά προβλήματα και 
		διορθώσεις για προβλήματα σχετικά με το ποντίκι 
                και το πληκτρολόγιο.

Msdosdrv.txt    Περιέχει πληροφορίες σύνταξης για MS-DOS προγράμματα
                οδήγησης συσκευών. Για πρόσθετη βοήθεια σχετικά με  
		εντολές MS-DOS, δείτε στο αρχείο Config.txt. 
		Μπορείτε επίσης να χρησιμοποιήσετε την βοήθεια της
		γραμμής εντολών, πληκτρολογώντας στην γραμμή εντολών 
		μετά το όνομα της εντολής τους χαρακτήρες "/?".
	
Network.txt     Παρέχει πληροφορίες για την εγκατάσταση και λειτουργία
		διακομιστών δικτύου.

Printers.txt    Παρέχει πληροφορίες για γνωστά προβλήματα και 
		διορθώσεις για προβλήματα σχετικά με τους εκτυπωτές.

Programs.txt    Παρέχει πληροφορίες και	διορθώσεις για την  
		εκτέλεση ειδικών προγραμμάτων Windows και MS-DOS
		με τα Windows 98.

Support.txt     Παρέχει πληροφορίες για το πως να λάβετε
		πρόσθετη υποστήριξη για τα Windows 98.

Tips.txt        Περιέχει μια συλλογή από συμβουλές και τεχνάσματα 
		για την χρήση των Windows 98, όπου τα περισσότερα
		από αυτά δεν αναφέρονται στην ηλεκτρονική Βοήθεια
		ή στο εγχειρίδιο.
