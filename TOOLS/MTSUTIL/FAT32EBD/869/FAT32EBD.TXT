=========
ΕΙΣΑΓΩΓΗ
=========

Η εφεδρική δισκέτα εκκίνησης FAT32 (EBD) παρέχει ένα μέσο προστατευμένο από ιούς
για την εκκίνηση και την αντιμετώπιση προβλημάτων ενός υπολογιστή, καθώς επίσης
και ένα μέσο για τη διαμόρφωση μεγάλων μονάδων δίσκων με τη βοήθεια του 
συστήματος αρχείων FAT32.

Το βοηθητικό πρόγραμμα το οποίο δημιουργεί την EBD, το FAT32EBD.EXE,
κάνει ένα αντίγραφο ενός τομέα εκκίνησης ο οποίος έχει εξεταστεί και
δεν περιέχει ιούς. Μια δισκέτα εκκίνησης χωρίς ιούς, μπορεί να χρησιμοποιηθεί
για την αντιμετώπιση προβλημάτων σε συστήματα για τα οποία έχετε την υποψία ότι 
έχουν προσβληθεί από ιό ή για τη δημιουργία ενός μεγάλου διαμερίσματος 
για τα Windows 98.


=========================================================
ΠΩΣ ΝΑ ΔΗΜΙΟΥΡΓΗΣΕΤΕ ΜΙΑ ΕΦΕΔΡΙΚΗ ΔΙΣΚΕΤΑ ΕΚΚΙΝΗΣΗΣ FAT32
=========================================================

Το βοηθητικό πρόγραμμα το οποίο δημιουργεί τη FAT32 EBD, το FAT32EBD.EXE, μπορεί να 
εκτελεστεί είτε από το MS-DOS είτε από τα Windows. Θα σας ζητηθεί να τοποθετήσετε
μια δισκέτα με χωρητικότητα 1.44 MB στη μονάδα δίσκου A:\ και μετά να συνεχίσετε.
Το FAT32EBD.EXE, μεταφέρει τον τομέα εκκίνησης στη δισκέτα ώστε να διασφαλιστεί
ότι δεν υπάρχουν ιοί. Στη συνέχεια, εκτελείται εξαγωγή των αρχείων που αφορούν το δίσκο
από τα αρχεία WIN98*.CAB, τα οποία μεταφέρονται στη δισκέτα.

------------
Απαιτήσεις
------------

 - Πρόσβαση στο CD-ROM των Windows 98
 - Πρόσβαση σε μια μονάδα δισκέτας με χωρητικότητα 1.44 MB 
   στη μονάδα δίσκου A:\
 - δισκέτα χωρητικότητας 1.44 MB

----------------------------------------------------------------
Βήματα για τη δημιουργία μιας εφεδρικής δισκέτας εκκίνησης FAT32
με χρήση των Windows 95 ή των Windows98
----------------------------------------------------------------

1. Κάντε κλικ στο κουμπί "Έναρξη", επιλέξτε την εντολή "Προγράμματα" και 
   στη συνέχεια κάντε κλικ στην Εξερεύνηση των Windows.

2. Επιλέξτε τη μονάδα του CD-ROM που περιέχει το CD των Windows 98.

3. Επιλέξτε και κάντε διπλό κλικ στο φάκελο \Tools, για να τον αναπτύξετε. 
   Αναπτύξτε τον υποκατάλογο \MTSutil και μετά τον υποκατάλογο \FAT32EBD. 
 
4. Κάντε διπλό κλικ στο αρχείο FAT32EBD.EXE και ακολουθήστε τις οδηγίες,
   για να δημιουργήσετε μια εφεδρική δισκέτα εκκίνησης FAT32.

5. Όταν ολοκληρωθεί η δημιουργία της εφεδρικής δισκέτας εκκίνησης, βεβαιωθείτε 
   ότι η δισκέτα διαθέτει προστασία εγγραφής, για να διασφαλίσετε ότι δεν θα
   μεταφερθούν ιοί στη δισκέτα.

6. Επανεκκινήστε τον υπολογιστή σας χρησιμοποιώντας την 
   εφεδρική δισκέτα εκκίνησης FAT32.

------------------------------------------------
Βήματα για τη δημιουργία μιας εφεδρικής δισκέτας 
εκκίνησης FAT32 με χρήση των Windows 3.x
------------------------------------------------

Για να έχετε καλύτερα αποτελέσματα, βγείτε από τα Windows και 
δημιουργήστε την εφεδρική δισκέτα εκκίνησης FAT32 χρησιμοποιώντας το MS-DOS. 
Όταν βγείτε από τα Windows, ανατρέξτε στη ακόλουθη ενότητα.

------------------------------------------------
Βήματα για τη δημιουργία μιας εφεδρικής δισκέτας 
εκκίνησης FAT32 με χρήση του MS-DOS
------------------------------------------------

1. Στη γραμμή εντολών εκτός των Windows, μεταβείτε στη μονάδα του CD-ROM που 
   περιέχει το CD των windows 98. Για παράδειγμα, εάν η μονάδα του CD-ROM που 
   διαθέτετε ονομάζεται D:\, θα πληκτρολογήσετε: 

	D:\    [πιέστε το πλήκτρο Enter]

2. Μεταβείτε στον κατάλογο FAT32EBD πληκτρολογώντας τα ακόλουθα:

	CD\TOOLS\MTSUTIL\FAT32EBD    [πιέστε το πλήκτρο Enter]

3. Εκτελέστε το αρχείο FAT32EBD.EXE και ακολουθήστε τις οδηγίες για τη δημιουργία
   μιας εφεδρικής δισκέτας εκκίνησης FAT32. Εκτελέστε το βοηθητικό πρόγραμμα
   πληκτρολογώντας:

	FAT32EBD.EXE    [πιέστε το πλήκτρο Enter]

4. Όταν ολοκληρωθεί η δημιουργία της εφεδρικής δισκέτας εκκίνησης, βεβαιωθείτε ότι
   η δισκέτα διαθέτει προστασία εγγραφής για να διασφαλίσετε ότι δεν θα μεταφερθούν
   ιοί στη δισκέτα.

5. Επανεκκινήστε τον υπολογιστή σας χρησιμοποιώντας την εφεδρική δισκέτα εκκίνησης FAT32.


===============================
ΔΗΜΙΟΥΡΓΙΑ ΜΟΝΑΔΩΝ ΔΙΣΚΩΝ FAT32
===============================

Στα Windows 98, εάν εκτελέσετε το εργαλείο FDISK σε ένα σύστημα το οποίο διαθέτει μονάδα
δίσκου χωρητικότητας μεγαλύτερης από 512 MB, θα ερωτηθείτε αν επιθυμείτε την ενεργοποίηση 
της υποστήριξης μεγάλων δίσκων. Εάν απαντήσετε Ναι, τότε οποιοδήποτε διαμέρισμα έχετε 
δημιουργήσει με μεγαλύτερη χωρητικότητα από 512 MB, θα επισημαίνεται ως διαμέρισμα FAT32.

   ΣΗΜΕΙΩΣΗ: Για να εκτελέσετε τον "Μετατροπέα μονάδας δίσκου (FAT32)", είναι απαραίτητη 
             η εγκατάσταση των Windows 98. Για περισσότερες πληροφορίες, ανατρέξτε στην 
             ηλεκτρονική Βοήθεια.


=============
ΓΝΩΣΤΑ ΘΕΜΑΤΑ
=============

Εάν αντιμετωπίσετε δυσκολίες κατά την εκτέλεση του αρχείου FAT32EBD.EXE, 
ανατρέξτε στην ακόλουθη λίστα γνωστών θεμάτων σχετικά με το εργαλείο.

------------------------------------------------------------
Αντιγραφή μη έγκυρων αρχείων στην εφεδρική δισκέτα εκκίνησης
------------------------------------------------------------

Εάν τα αρχεία στην εφεδρική δισκέτα εκκίνησης FAT32 έχουν μέγεθος 
0 (μηδέν) byte, αυτό ενδέχεται να είναι αποτέλεσμα του ορισμού της 
μεταβλητής περιβάλλοντος TMP στη μονάδα δισκέτας ή ανεπαρκούς συμβατικής 
μνήμης.

Ορισμός μεταβλητής TMP στη μονάδας δισκέτας
-------------------------------------------

Η εκκίνηση του υπολογιστή σας με μια δισκέτα, ενδέχεται να ορίσει τη μεταβλητή
περιβάλλοντος TMP στη μονάδα δισκέτας. Το αρχείο FAT32EBD.EXE απαιτεί η 
μεταβλητή περιβάλλοντος TMP να έχει οριστεί στη μονάδα σκληρού δίσκου. 
Εάν η μεταβλητή TMP έχει οριστεί ως μονάδα δισκέτας, ενδέχεται τότε το αρχείο 
FAT32EBD.EXE να δημιουργήσει μια δισκέτα εκκίνησης με αρχεία
μεγέθους μηδέν byte. 

Για να προσδιορίσετε την τρέχουσα τιμή της μεταβλητής TMP, πληκτρολογήστε:

	SET	[πιέστε το πλήκτρο Enter]

Για να ορίσετε τη μεταβλητή περιβάλλοντος TMP στον σκληρό δίσκο, πληκτρολογήστε:

	SET TMP=C:\	[πιέστε το πλήκτρο Enter]

Ανεπαρκής συμβατική μνήμη
-------------------------

Το αρχείο FAT32EBD.EXE βασίζεται στο βοηθητικό πρόγραμμα των Windows 98 EXTRACT.EXE
Το EXTRACT απαιτεί περίπου 520K ελεύθερης συμβατικής μνήμης. 
Η φόρτωση προγραμμάτων οδήγησης CD-ROM πραγματικής κατάστασης λειτουργίας και
συμπίεσης Συμπίεση Δίσκων/DoubleSpace ενδέχεται να μειώσει την ποσότητα συμβατικής 
μνήμης. Για να μην συμβεί αυτό, τροποποιήστε τα αρχεία CONFIG.SYS και
AUTOEXEC.BAT, ώστε να φορτώνονται οι συσκευές στην υψηλή μνήμη.

   ---- Παράδειγμα CONFIG.SYS ----

   Device=C:\windows\himem.sys
   Device=C:\windows\Emm386.exe noems
   Dos=high,umb
   Devicehigh=C:\CDROM\oakcdrom.sys /d:mscd001
   Devicehigh=C:\Windows\drvspace.sys /move

   ---- Παράδειγμα AUTOEXEC.BAT ----

   LH C:\CDROM\mscdex.exe /d:mscd001

----------------------------------
Ακατάλληλη ή κατεστραμμένη δισκέτα
----------------------------------

Εάν κατά την εκτέλεση του αρχείου FAT32EBD.EXE εμφανιστούν μηνύματα σφάλματος όπως
"Σφάλμα εγγραφής" ή "Δεν είναι δυνατή η εγγραφή της δισκέτα στη μονάδα δίσκου A:\",
ενδέχεται να χρησιμοποιείτε μια κατεστραμμένη δισκέτα. 

----------------------------------------------------
Χαμηλή επίδοση κατά τη δημιουργία δισκέτας εκκίνησης
----------------------------------------------------

Το FAT32EBD.EXE εξάγει αρχεία για την εφεδρική δισκέτα εκκίνησης FAT32 από το CD των
Windows 98. Η ταχύτητα εκτέλεσης του EXTRACT.EXE εκτός περιβάλλοντος των 
Windows ενδέχεται να μειωθεί. Για να βελτιώσετε την ταχύτητα εκτέλεσης του  
FAT32EBD.EXE εκτός του περιβάλλοντος των Windows, φορτώστε το SMARTDRV.EXE. 

   ΣΗΜΕΙΩΣΗ: Για να φορτώσετε το SMARTDRV.EXE των Windows 98 μεταβείτε στο φάκελο
	     \TOOLS\OLDMSDOS στο CD των Windows 98.

--------------------------------
Δισκέτα εκκίνησης των Windows 98
--------------------------------

Η εφεδρική δισκέτα εκκίνησης FAT32 EBD δεν περιλαμβάνει όλα τα εργαλεία 
αντιμετώπισης προβλημάτων που είναι διαθέσιμα στη δισκέτα εκκίνησης 
των Windows 98. Η Microsoft προτείνει τη δημιουργία μιας δισκέτας 
εκκίνησης των Windows 98 κατά τη διάρκεια της εγκατάστασης των
Windows 98 ή από την καρτέλα "Δισκέτα εκκίνησης" της επιλογής 
"Προσθαφαίρεση προγραμμάτων" του Πίνακα Ελέγχου. 


========================
ΠΕΡΙΣΣΟΤΕΡΕΣ ΠΛΗΡΟΦΟΡΙΕΣ
========================

Ένας ιός υπολογιστή είναι ένα εκτελέσιμο αρχείο το οποίο είναι σχεδιασμένο
να αναπαράγει τον εαυτό του και να αποφεύγει τον εντοπισμό. Ένας ιός στην 
προσπάθειά του να αποφύγει τον εντοπισμό ενδέχεται να εμφανιστεί ως ένα 
κανονικό πρόγραμμα. Η επανεγγραφή και η ρύθμιση είναι δύο τρόποι που εφαρμόζουν
οι ιοί, για να αποφύγουν τον εντοπισμό. Η συνεχής ενημέρωση των προγραμμάτων 
εντοπισμού ιών είναι απαραίτητη, ώστε τα προγράμματα να έχουν τη δυνατότητα 
να εντοπίζουν τους νέους και τροποποιημένους ιούς.

Κατά την εκκίνηση ενός υπολογιστή εξετάζεται πρώτα ο τομέας εκκίνησης του 
σκληρού δίσκου, πριν να φορτωθεί το λειτουργικό σύστημα ή οποιοδήποτε 
άλλο αρχείο εκκίνησης. Ένας ιός του τομέα εκκίνησης είναι σχεδιασμένος έτσι 
ώστε να αντικαθιστά τις πληροφορίες των τομέων εκκίνησης του σκληρού δίσκου 
με αυτές του δικού του κώδικα. Όταν ο υπολογιστής έχει προσβληθεί από έναν ιό 
του τομέα εκκίνησης, τότε πριν από οποιαδήποτε άλλη ενέργεια γίνεται εγγραφή
του κώδικα του ιού στη μνήμη. Από τη στιγμή που βρεθεί στη μνήμη, μπορεί να 
αναπαραγάγει τον κώδικά του σε άλλους δίσκους που χρησιμοποιούνται από τον
προσβεβλημένο υπολογιστή.

-------------------
Περιγραφή του FAT32
-------------------

Τα Windows 98 παρέχουν μια ενημερωμένη έκδοση του συστήματος αρχείων του 
πίνακα εκχώρησης αρχείων που ονομάζεται FAT32. Με αυτό το ενημερωμένο 
σύστημα αρχείων, είναι δυνατή η υποστήριξη προεπιλεγμένου μεγέθους συμπλέγματος
έως 4K, καθώς επίσης και η υποστήριξη μεγέθους σκληρών
δίσκων μεγαλύτερο από 2 GB.

 - Δεν είναι δυνατή η συμπίεση ενός τόμου FAT32 μέσω της λειτουργίας 
   Συμπίεση Δίσκων της Microsoft ή της Συμπίεσης Δίσκων 3.

 - Δεν είναι δυνατή η πρόσβαση στους τόμους FAT32 αν γίνει εκκίνηση του 
   υπολογιστή με άλλο λειτουργικό σύστημα. 

 - Ενδέχεται να μην είναι δυνατή η αναγνώριση ενός τόμου FAT32 από το MS-DOS, 
   Windows NT 3.x και 4.x, Windows 1.x, 2.x, και 3.x καθώς επίσης τα Windows 95 (950).

   ΣΗΜΕΙΩΣΗ: Η ενημερωμένη έκδοση των Windows 95 OEM Service Release 2 (1111)
             υποστηρίζει το FAT32.

Συγκριτικά με τις προηγούμενες εφαρμογές του συστήματος αρχείων FAT, το FAT32 παρέχει
τις ακόλουθες βελτιώσεις:

 - Υποστηρίζει μονάδες δίσκων χωρητικότητας μέχρι και 2 terabyte.

 - Το FAT32 χρησιμοποιεί μικρότερα συμπλέγματα (δηλ. συμπλέγματα μεγέθους 4K
   για μονάδες δίσκων μεγέθους έως 8 GB) με αποτέλεσμα μια πιο αποδοτική χρήση 
   του χώρου στο δίσκο, συγκριτικά με τις μεγάλες μονάδες δίσκων FAT16.

 - Το FAT32 έχει τη δυνατότητα να μεταθέτει τον ριζικό κατάλογο και να
   χρησιμοποιεί το εφεδρικό αντίγραφο του FAT αντί του προεπιλεγμένου αντιγράφου.
   Επιπλέον, έχει γίνει επέκταση της εγγραφής εκκίνησης των μονάδων δίσκων FAT32
   ώστε να συμπεριλαμβάνουν ένα εφεδρικό αντίγραφο για κρίσιμες δομές δεδομένων.
   Αυτό σημαίνει ότι οι μονάδες δίσκων FAT32 είναι λιγότερο επιρρεπείς σε αποτυχία 
   (γνωστή ως SPOF) που οφείλεται στην ύπαρξη μόνο ενός πίνακα εκχώρησης αρχείων 
   συγκριτικά με τους ήδη υπάρχοντες τόμους FAT16.

 - Ο ριζικός κατάλογος σε μια μονάδα δίσκου FAT32 είναι πλέον μια κανονική αλυσίδα
   συμπλεγμάτων, ώστε να έχει τη δυνατότητα να βρίσκεται σε οποιαδήποτε θέση στη 
   μονάδα δίσκου. Για αυτόν το λόγο όσοι περιορισμοί υπήρχαν για τον αριθμό καταχωρήσεων
   στο ριζικό κατάλογο έχουν εξαλειφθεί.
