Υπηρεσία απομακρυσμένου μητρώου της Microsoft (REGSERV.EXE και σχετικά αρχεία)

Εγκαταστήστε αυτό το λογισμικό σε υπολογιστές με Windows 95 ή Windows 98 για 
να επιτρέπετε στους διαχειριστές να αλλάζουν τις εγγραφές του μητρώου σε 
υπολογιστές μέσω του δικτύου.

Οι τοπικοί και απομακρυσμένοι υπολογιστές  πρέπει να εκτελούν όλοι 
την υπηρεσία απομακρυσμένου μητρώου και η ασφάλεια του υπολογιστή που 
διαχειρίζεστε, σε επίπεδο χρήστη, πρέπει να είναι ενεργοποιημένη. 
Για να αλλάξετε το μητρώο, μπορείτε να χρησιμοποιήσετε την επεξεργασία 
μητρώου ή την επεξεργασία πολιτικής του συστήματος.

Για να εγκαταστήσετε αυτήν την υπηρεσία, χρησιμοποιήστε το εικονίδιο "Δίκτυο" 
στον  Πίνακα Ελέγχου, κάντε κλικ στο κουμπί "Προσθήκη", επιλέξτε το στοιχείο 
δικτύου "Υπηρεσία", κάντε κλικ στο κουμπί "Προσθήκη", κάντε κλικ στο κουμπί 
"Από δισκέτα" και εγκαταστήστε την από τον κατάλογο TOOLS\RESKIT\NETADMIN\REMOTREG.

Για περισσότερες πληροφορίες για το απομακρυσμένο μητρώο, δείτε στην παράγραφο 
του Windows 98 Resource Kit ή στην ηλεκτρονική βοήθεια εργαλείων του Windows 98 
Resource Kit (WIN98RK.HLP). Μπορείτε να έχετε πρόσβαση σε αυτή την βοήθεια 
εγκαθιστώντας το δείγμα εργαλείων του Windows 98 Resource Kit από τον κατάλογο 
TOOLS\RESKIT του CD των Windows 98.
