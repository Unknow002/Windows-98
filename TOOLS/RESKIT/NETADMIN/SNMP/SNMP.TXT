Παράγων SNMP (SNMP.EXE και σχετικά αρχεία)

Χρησιμοποιήστε αυτό το λογισμικό για να εποπτεύσετε απομακρυσμένες συνδέσεις 
σε υπολογιστές με Windows 95 ή Windows 98, αν το δίκτυό σας, για την διαχείρισης
του συστήματος χρησιμοποιεί προϊόντα διαχείρισης συστήματος Simple Network 
Management Protocol (SNMP). 

Ο παράγων SNMP των Windows 95 συμφωνεί με τα χαρακτηριστικά του SNMP έκδοση 1.
Αυτός ο παράγοντας σας επιτρέπει να εποπτεύετε, από μια κονσόλα SNMP, 
απομακρυσμένες συνδέσεις σε υπολογιστές με Windows 95 ή Windows 98. Όταν αυτός 
ο παράγοντας εγκατασταθεί, δεν χρειάζεται να κάνετε καμία τροποποίηση στους 
υπολογιστές-πελάτες για να χρησιμοποιήσετε το SNMP. Ο παράγοντας SNMP έχει 
υλοποιηθεί ως υπηρεσία βασισμένη σε Win32 και λειτουργεί χρησιμοποιώντας 
υποδοχές των Windows μέσω και TCP/IP και IPX/SPX πρωτοκόλλων. 
Οι παράγοντες επέκτασης έχουν υλοποιηθεί ως Win32 DLL.

Για να εγκαταστήσετε τον παράγοντα SNMP σε ένα υπολογιστή με Windows 98, 
χρησιμοποιήστε το εικονίδιο "Δίκτυο" στον  Πίνακα Ελέγχου, κάντε κλικ στο 
κουμπί "Προσθήκη", επιλέξτε το στοιχείο δικτύου "Υπηρεσία", κάντε κλικ στο 
κουμπί "Προσθήκη", κάντε κλικ στο κουμπί "Από δισκέτα" και εγκαταστήστε τον 
από τον κατάλογο TOOLS\RESKIT\NETADMIN\SNMP.

Για περισσότερες πληροφορίες για τον παράγοντα εποπτείας δικτύου, δείτε στην
παράγραφο του Windows 98 Resource Kit ή στην ηλεκτρονική βοήθεια εργαλείων 
του Windows 98 Resource Kit (WIN98RK.HLP). Μπορείτε να έχετε πρόσβαση σε αυτή 
την βοήθεια εγκαθιστώντας το δείγμα εργαλείων του Windows 98 Resource Kit από 
τον κατάλογο TOOLS\RESKIT του CD των Windows 98.
