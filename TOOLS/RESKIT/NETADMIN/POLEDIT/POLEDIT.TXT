Επεξεργασία πολιτικής συστήματος

Χρησιμοποιήστε αυτό το εργαλείο για να δημιουργήσετε ή να επεξεργαστείτε 
τις πολιτικές του συστήματος για να τυποποιήσετε την εμφάνιση και τις 
δυνατότητες των Windows 95 ή Windows 98 για ένα μόνο χρήστη, μιας ομάδας 
χρηστών ή όλου του δικτύου. 

Μπορείτε να δημιουργήσετε νέα αρχεία πολιτικής ή να χρησιμοποιήσετε τα 
αρχεία δείγματα (.POL) τα οποία περιέχονται στον φάκελο 
TOOLS\RESKIT\NETADMIN\POLEDIT στο αποθηκευτικό μέσο εγκατάστασης.

Για να εγκαταστήσετε αυτό το εργαλείο στον τοπικό σας σκληρό δίσκο ή για 
να εγκαταστήσετε υποστήριξη για πολιτικές ομάδων, χρησιμοποιήστε την 
επιλογή "Προσθαφαίρεση προγραμμάτων" του Πίνακα Ελέγχου. Επιλέξτε την 
καρτέλα "Εγκατάσταση των Windows", κάντε κλικ στο κουμπί "Από δισκέτα" 
και εγκαταστήστε το από τον κατάλογο TOOLS\RESKIT\NETADMIN\POLEDIT που 
βρίσκεται στο αποθηκευτικό μέσο εγκατάστασης.

Περισσότερες πληροφορίες για τις πολιτικές συστήματος και την επεξεργασία 
πολιτικής συστήματος θα βρείτε στις παραγράφους του Windows 95 Resource Kit 
(WIN95RK.HLP) ή Windows 98 Resource Kit (WIN98RK.HLP).
