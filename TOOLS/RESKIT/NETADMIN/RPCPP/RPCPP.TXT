Microsoft RPC Print Provider (RPCPP)

Η κλήση απομακρυσμένης διαδικασίας (RPC) του Microsoft Print Provider 
βελτιώνει τις εκτυπώσεις δικτύου και την απομακρυσμένη διαχείριση με 
την παροχή του συνόλου των API τύπου Win32 που απαιτούνται για ένα 
υπολογιστή-πελάτη με Windows 95 ή Windows 98 για να γίνεται απομακρυσμένη 
παύση και ακύρωση εκτυπώσεων σε διακομιστές Windows NT. 
Χρησιμοποιώντας αυτή την παροχή εκτυπώσεων , ένας υπολογιστής-πελάτης με 
Windows 95 ή Windows 98 μπορεί να αποκτήσει πληροφορίες για την κατάσταση 
λογαριασμού ή εκτυπώσεων από τον διακομιστή των Windows NT. 

Για να εγκαταστήσετε αυτό το λογισμικό στον υπολογιστή σας, χρησιμοποιήστε 
την επιλογή "Δίκτυο" στον  Πίνακα Ελέγχου, κάντε κλικ στο κουμπί "Προσθήκη", 
επιλέξτε το στοιχείο δικτύου "Υπηρεσία", κάντε κλικ στο κουμπί "Προσθήκη", 
κάντε κλικ στο κουμπί "Από δισκέτα" και εγκαταστήστε τον από τον κατάλογο 
TOOLS\RESKIT\NETADMIN\RPCPP.

Για περισσότερες πληροφορίες για το RPCPP, δείτε στην παράγραφο του 
Windows 98 Resource Kit ή στην ηλεκτρονική βοήθεια εργαλείων του Windows 98 
Resource Kit (WIN98RK.HLP). Μπορείτε να έχετε πρόσβαση σε αυτή την βοήθεια 
εγκαθιστώντας το δείγμα εργαλείων του Windows 98 Resource Kit από τον κατάλογο 
TOOLS\RESKIT του CD των Windows 98.
