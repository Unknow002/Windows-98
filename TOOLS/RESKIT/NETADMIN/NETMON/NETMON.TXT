Παράγων εποπτείας δικτύου (NMAGENT.* και σχετικά αρχεία)

Ο φάκελος NETMON περιέχει το λογισμικό για ένα πρόγραμμα οδήγησης 
πρωτοκόλλου και τον παράγοντα εποπτείας των δικτύων της Microsoft. 
Το πρόγραμμα οδήγησης πρωτοκόλλου παρέχει μετρητές επιδόσεων οι οποίοι
μπορούν να προβληθούν χρησιμοποιώντας την Εποπτεία Συστήματος αν 
θέλετε να έχετε πρόσβαση σε συγκεκριμένα στατιστικά κυκλοφορίας δικτύου 
για προσαρμογείς δικτύου προστατευμένης λειτουργίας NDIS 3.1.

Ο παράγοντας εποπτείας δικτύου, ο οποίος μπορεί να εκτελεστεί σαν 
υπηρεσία και των Windows 95 και των Windows 98, λειτουργεί με το 
πρόγραμμα οδήγησης πρωτοκόλλου για να χρησιμοποιηθεί με την εφαρμογή 
εποπτείας των δικτύων της Microsoft. Μπορείτε να χρησιμοποιήσετε αυτήν 
την εφαρμογή για να εντοπίσετε και να επιλύσετε προβλήματα σε δίκτυα 
LAN, WAN και συνδέσεις Remote Access Service (RAS) της Microsoft.

ΣΗΜΕΙΩΣΗ: Η εφαρμογή Εποπτεία Δικτύου παρέχεται με τον Microsoft 
Systems Management Server, ο οποίος είναι σύστημα διακομιστή 
προγρογράμματος-πελάτη που επιτρέπει στους διαχειριστές να εκτελούν 
βασικές λειτουργίες διαχείρισης για απομακρυσμένους υπολογιστές από 
μια κεντρική τοποθεσία. 

Για να χρησιμοποιήσετε τον παράγοντα Εποπτείας Δικτύου για να επιλύσετε 
προβλήματα στο δίκτυο, πρέπει να τον εγκαταστήσετε (NMAGENT.EXE) σε 
κάθε υπολογιστή με τον οποίο θα εποπτεύετε το δίκτυο.

Για να εγκαταστήσετε τον παράγοντα Εποπτείας Δικτύου σε ένα υπολογιστή 
με  Windows 95 ή Windows 98, χρησιμοποιήστε το εικονίδιο "Δίκτυο" στον 
Πίνακα Ελέγχου, κάντε κλικ στο κουμπί "Προσθήκη", επιλέξτε το στοιχείο 
δικτύου "Υπηρεσία", κάντε κλικ στο κουμπί "Προσθήκη", κάντε κλικ στο 
κουμπί "Από δισκέτα" και εγκαταστήστε τον από τον κατάλογο 
ΤOOLS\RESKIT\NETADMIN\NETMON.
 
Μπορείτε επίσης να εγκαταστήσετε τον Παράγοντα Εποπτείας Δικτύου 
χρησιμοποιώντας εντολές στην δέσμη ενεργειών του προγράμματος 
Εγκατάστασης. Για περισσότερες πληροφορίες για τον παράγοντα Εποπτείας 
Δικτύου, δείτε στην παράγραφο του Windows 98 Resource Kit ή στην 
ηλεκτρονική βοήθεια εργαλείων του Windows 98 Resource Kit 
(WIN98RK.HLP). Μπορείτε να έχετε πρόσβαση σε αυτήν την βοήθεια 
εγκαθιστώντας το δείγμα εργαλείων του Windows 98 Resource Kit από 
τον κατάλογο TOOLS\RESKIT του CD των Windows 98.
