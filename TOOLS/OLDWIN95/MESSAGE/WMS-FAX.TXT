ΤΙ ΕΙΝΑΙ ΤΟ MICROSOFT EXCHANGE; 
==============================

Το Microsoft Exchange σάς επιτρέπει να στέλνετε και να λαμβάνετε ηλεκτρονικό 
ταχυδρομείο, φαξ και έγγραφα σε μηνύματα με πρόσβαση σε ένα λογαριασμό Microsoft 
Mail Postoffice Mailbox Account. Στις υπηρεσίες πληροφοριών που μπορούν
να χρησιμοποιηθούν με το Microsoft Exchange συμπεριλαμβάνονται το Microsoft Mail,
Internet Mail, Φαξ της Microsoft και το Microsoft Network. Το Microsoft Exchange 
είναι ένα πλήρη πρόγραμμα-πελάτης MAPI.

ΤΙ ΕΙΝΑΙ ΤΟ MICROSOFT MAIL POSTOFFICE;
======================================

Το Microsoft Mail Postoffice συμπεριλαμβανόταν στα Windows for Workgroups, 
Windows 95, OSR2 και OSR2.5. Το Microsoft Mail Postoffice προσφέρει μια κεντρική 
τοποθεσία για τους Διαχειριστές Δικτύου ή Διαχειριστές Small Office για να 
δημιουργούν και να διατηρούν μια Workgroup Postoffice.  Ο Διαχειριστής θα 
δημιουργεί ένα λογαριασμό για κάθε χρήστη της the Workgroup Postoffice.  Αυτός
ο λογαριασμός θα προσφέρει στους χρήστες την δυνατότητα να στέλνουν και να 
λαμβάνουν ηλεκτρονικό ταχυδρομείο από ή σε άλλους χρήστες του δικτύου της εταιρίας. 

ΤΙ ΕΙΝΑΙ ΤΟ ΦΑΞ ΤΗΣ MICROSOFT;
==============================

Το Φαξ της Microsoft είναι μια εφαρμογή φαξ η οποία απαιτεί ένα πλήρες 
πρόγραμμα-πελάτη ΜAPI για να λειτουργήσει, όπως τα: Microsoft Exchange, 
Microsoft Exchange Server Client ή Outlook97. Το Φαξ της Microsoft αρχικά 
συμπεριλαμβανόταν στα Windows for Workgroups ως At Work Fax και αργότερα ως 
ενημερωμένο Φαξ της Microsoft Fax και συμπεριλαμβανόταν στα Windows 95, OSR2 
και OSR2.5.  Αυτή είναι η ίδια ενημερωμένη έκδοση και δεν περιλαμβάνει καμία
αλλαγή χαρακτηριστικών.  Το Φαξ της Microsoft απαιτεί μια συσκευή fax modem.



ΕΓΚΑΤΑΣΤΑΣΗ
===========
WMS.EXE

Εντοπίστε το αρχείο WMS.EXE στο CD ROM των Windows 98 στον κατάλογο 
Tools\OldWin95\Message. Κάντε διπλό κλικ για να εκτελέστε το αρχείο ή επιλέξτε το
και πατήστε Εnter. Ακολουθήστε τις ερωτήσεις του προγράμματος εγκατάστασης. Αν σας
ζητηθεί να εκτελέσετε επανεκκίνηση, τότε η εγκατάσταση έχει ολοκληρωθεί. Επιλέξτε
το κουμπί "OK" για επανεκκίνηση και ολοκλήρωση των ρυθμίσεων. Όταν ολοκληρωθεί η 
εγκατάσταση, οι χρήστες για πρώτη φορά πρέπει να δημιουργήσουν ένα προφίλ.
Μπορείτε να δημιουργήσετε ένα νέο προφίλ με διπλό κλικ στο εικονίδιο "Εισερχόμενα"
στην Επιφάνεια Εργασίας ή επιλέγοντας το εικονίδιο "Εισερχόμενα" στην Επιφάνεια
Εργασίας και πατώντας Enter. Ακολουθήστε τις οδηγίες του οδηγού εγκατάστασης "Εισερχόμενα".

Αυτό το αρχείο εγκαθιστά το Microsoft Exchange και το Microsoft Mail Postoffice, το οποίο 
συχνά αναφέρεται ως Workgroup Postoffice. Το Microsoft Exchange είναι ένα πλήρες 
πρόγραμμα-πελάτης MAPI. Το αρχείο WMS.EXE προσφέρεται για χρήστες του Microsoft Technical 
Support (MTS), ή χρήστες αναβάθμισης των Windows 95, OSR2 και OSR2.5 για να επιδιορθώσουν 
ήδη υπάρχοντα προγράμματα-πελάτες του Microsoft Exchange των Windows 95, OSR2 και OSR2.5 και 
Microsoft Mail Postoffice, αν καταστραφούν. (Ο χαρακτηρισμός "Ήδη υπάρχοντα προγράμματα-πελάτες"
απαιτεί την πλήρη λειτουργικότητα των Microsoft Exchange και Microsoft Mail Postoffice πριν 
την αναβάθμιση σε Windows 98.)  Αν καταστραφούν τα αρχεία του Microsoft Exchange, αυτό το αρχείο
θα εγκαταστήσει ενημερωμένες εκδόσεις αυτών των αρχείων. Το αρχείο WMS.EXE θα προσθέσει ένα
εικονίδιο "Εισερχόμενα" στην Επιφάνεια Εργασίας καθώς και ένα εικονίδιο "Αλληληγραφία" και
εικονίδιο "Microsoft Mail Postoffice" στον Πίνακα Ελέγχου, αν δεν υπάρχει ήδη. Αν το At Work Fax 
είχε ρυθμιστεί στο Microsoft Mail των Windows For Workgroups πριν την αναβάθμιση στα Windows 98, 
αυτό το αρχείο θα καταργήσει την λειτουργικότητα του "At Work Fax" στο Microsoft Mail και θα 
αφαιρέσει το εικονίδιο "At Work Fax" από τον Πίνακα Ελέγχου. Το εικονίδιο "Εισερχόμενα" στην 
Επιφάνεια Εργασίας θα ανοίξει στο Microsoft Exchange μόλις δημιουργηθεί ένα προφίλ. Το σύστημα 
πρέπει να έχει πρόσβαση σε λογαριασμό του Workgroup Postoffice για να μπορεί να δημιουργήσει ένα προφίλ.

ΕΙΔΟΠΟΙΗΣΗ: Αυτό το αρχείο δεν περιλαμβάνει καμία αλλαγή χαρακτηριστικών του Μicrosoft Exchange 
ή του Microsoft Mail Postoffice. Όλα τα γνωστά θέματα σχετικά με το Microsoft Exchange και 
Microsoft Mail Postoffice έχουν υλοποιηθεί σε αυτή την ενημερωμένη έκδοση. Αυτό το αρχείο 
δεν προτείνεται για χρήση με Microsoft Exchange Server Client ή Outlook 97. Αν παρουσιαστούν 
σφάλματα με τον Microsoft Exchange Server Client ή Outlook 97, προτείνεται να εγκαταστήσετε πάλι
την αρχική εφαρμογή. Αν είναι αναγκαίο να εγκαταστήσετε το αρχείο WMS.EXE για να αποκαταστήσετε
την λειτουργικότητα του Microsoft Mail Postoffice, προτείνεται να εγκαταστήσετε πάλι το Microsoft
Exchange Server Client ή Outlook 97 αφού ολοκληρωθεί η εγκατάσταση του αρχείου WMS.EXE.



AWFAX.EXE

Εντοπίστε το αρχείο AWFAX.EXE στο CD ROM των Windows 98 στον κατάλογο 
Tools\OldWin95\Message. Κάντε διπλό κλικ για να εκτελέστε το αρχείο ή επιλέξτε το
και πατήστε Εnter. Ακολουθήστε τις ερωτήσεις του προγράμματος εγκατάστασης. Θα σας
ζητηθεί να εκτελέσετε επανεκκίνηση, όταν η εγκατάσταση έχει ολοκληρωθεί. Επιλέξτε
το κουμπί "OK" για επανεκκίνηση και ολοκλήρωση των ρυθμίσεων.
Προσθέστε το Φαξ της Microsoft ως υπηρεσία στο προφίλ του προγράμματος-πελάτη MAPI 
μέσω των εικονιδίων "Αλληλογραφία" και "Φαξ" του Πίνακα Ελέγχου.

Το αρχείο ΑWFAX.EXE εγκαθιστά το Φαξ της Microsoft. Το Φαξ της Microsoft απαιτεί ένα 
πλήρες πρόγραμμα-πελάτης MAPI για να λειτουργεί, όπως: Microsoft Exchange, Microsoft Exchange 
Server Client ή Outlook 97. Το αρχείο AWFAX.EXE προσφέρεται για χρήστες του Microsoft Technical 
Support (MTS) ή χρήστες αναβάθμισης των Windows 95, OSR2 και OSR2.5 για να επιδιορθώσουν 
ήδη υπάρχουσα εγκατάσταση του Φαξ της Microsoft αν καταστραφεί. (Ο χαρακτηρισμός "Ήδη υπάρχουσα" 
εγκατάσταση απαιτεί την πλήρη λειτουργικότητα του Φαξ της Microsoft πριν την αναβάθμιση σε 
Windows 98.) Αν τα αρχεία του Φαξ της Microsoft καταστραφούν, αυτό το αρχείο θα εγκαταστήσει 
ενημερωμένες εκδόσεις αυτών των αρχείων. Το αρχείο AWFAX.EXE θα προσθέσει ένα εκτυπωτή Φαξ της Microsoft
στον φάκελο "Εκτυπωτές", αν δεν υπάρχει κάποιος ήδη. Αν το "At Work Fax" είχε ρυθμιστεί στο 
Windows For Workgroups Microsoft Mail πριν την αναβάθμιση στα Windows 98, αυτό το αρχείο θα καταργήσει 
την λειτουργικότητα του "At Work Fax" στο Microsoft Mail και θα αφαιρέσει το εικονίδιο "At Work Fax" 
από τον Πίνακα Ελέγχου.
Το Φαξ της Microsoft πρέπει να είναι μια υπηρεσία που έχει προστεθεί στο προφίλ του προγράμματος-πελάτης
MAPI για να λειτουργεί.

ΕΙΔΟΠΟΙΗΣΗ: Αυτό το αρχείο δεν περιλαμβάνει καμία αλλαγή χαρακτηριστικών του Φαξ της Μicrosoft. 
Όλα τα γνωστά θέματα σχετικά με το Φαξ της Microsoft έχουν υλοποιηθεί σε αυτή την ενημερωμένη έκδοση. 

Για να απαντήσει το Φαξ της Microsoft σε μια εισερχόμενη κλήση και να λάβει ένα φαξ, πρέπει να έχει 
φορτωθεί και να εκτελείτε το πρόγραμμα-πελάτης MAPI.



ΚΑΤΑΡΓΗΣΗ ΕΓΚΑΤΑΣΤΑΣΗΣ
======================
 
Για να καταργήσετε κάποια εγκατάσταση κάποιου από αυτά τα προγράμματα πηγαίνετε στον Πίνακα Ελέγχου
και κάντε διπλό κλικ στην "Προσθαφαίρεση προγραμμάτων". Το στοιχείο θα βρίσκεται στην λίστα της πρώτης 
καρτέλας, Εγκατάσταση/κατάργηση. Επιλέξτε το πρόγραμμα που επιθυμείτε να καταργήσετε την εγκατάστασή του
και κάντε κλικ στο κουμπί "Προσθαφαίρεση".


ΧΡΗΣΗ ΤΟΥ MICROSOFT EXCHANGE ΜΕ ΤΟ MICROSOFT SCHEDULE+ 1.0
=========================================================

Το Microsoft Exchange έχει σχεδιαστεί για να συνεργάζεται με το 
Microsoft Schedule+ 7.0. Αφού εγκατασταθεί το Microsoft Exchange θα 
απενεργοποιήσει μερικώς το Microsoft Schedule+ 1.0. Θα μπορείτε ακόμη 
να χρησιμοποιείτε το Microsoft Schedule+ 1.0 σε κατάσταση αυτόνομης 
λειτουργίας, αλλά δεν θα μπορείτε να στέλνετε ή να λαμβάνετε προσκλήσεις 
σε σύσκεψη. Όταν εκκινείτε το Microsoft Schedule+ 1.0 θα δείτε ένα μήνυμα 
σφάλματος που σας ειδοποιεί ότι δεν είναι δυνατή η εκκίνηση του προγράμματος 
για αποστολή ταχυδρομείου σε ουρά. Κάντε κλικ στο κουμπί "Ναι" για να 
προχωρήσετε και το Microsoft Schedule+ 1.0 θα συνεχίσει. 

Το Microsoft Schedule+ θα απενεργοποιηθεί αν αλλάξετε το Mailspl.exe σε 
Mailspl.bak και εισαγάγετε τη γραμμή "StartupOffline=1" στο Schdplus.ini. 

ΣΗΜΕΙΩΣΗ:  Δεν υπάρχει μήνυμα που σας ειδοποιεί ότι έγιναν αυτές οι 
αλλαγές. 


ΧΡΗΣΗ ΤΟΥ MICROSOFT EXCHANGE ΜΕ ΤΟ MICROSOFT MAIL
=================================================

Το Microsoft Exchange έχει σχεδιαστεί για να αντικαθιστά το πρόγραμμα 
Microsoft Mail που περιλαμβάνεται στην έκδοση 3.χ των Windows for Workgroups
και του Microsoft Mail. Το αρχείο WMS.EXE περιλαμβάνει τον μετατροπέα MMF
ο οποίος σας επιτρέπει να εισάγετε τα μηνύματα του Microsoft Mail στο 
Microsoft Exchange. 


ΣΥΜΒΟΥΛΕΣ ΓΙΑ ΤΗΝ ΑΝΤΙΜΕΤΩΠΙΣΗ ΠΡΟΒΛΗΜΑΤΩΝ
==========================================

Εάν έχετε προβλήματα σύνδεσης με το ταχυδρομικό γραφείο σας του 
Microsoft Mail, βεβαιωθείτε ότι έχετε πρόσβαση δικτύου στο ταχυδρομικό 
γραφείο σας. Μπορείτε επίσης να λάβετε λεπτομέρειες διαγνωστικών για την
αιτία των προβλημάτων από το Microsoft Exchange με κλικ στο μενού "Εργαλεία", 
εντολή "Εργαλεία του Microsoft Mail" και στην καρτέλα "Προβολή του αρχείου 
καταγραφής περιόδου λειτουργίας". 

Εάν έχετε πρόβλημα σύνθεσης ή ανάγνωσης μηνύματος που λέει "Δεν είναι 
δυνατή η εκκίνηση φόρμας", τότε ίσως είναι καταστραμμένο το αρχείο με 
τις φόρμες. Για να λύσετε το πρόβλημα, κλείστε το Microsoft Exchange 
και περάστε στον κατάλογο {Windows}\Forms και διαγράψτε το αρχείο 
Frmcache.dat. Από το {Windows}\Forms αντιγράψτε το αρχείο Frmcache.bak 
στο Frmcache.dat και επανεκκινήστε το Microsoft Exchange. 

Μερικές επεκτάσεις του Microsoft Mail 3.x δεν είναι συμβατές με το 
Microsoft Exchange--για παράδειγμα, WinRules και Conference+.  Το
Microsoft Exchange αυτόματα απενεργοποιεί μερικές από αυτές, για να αποτρέψει
προβλήματα του συστήματος. Εάν μια επέκταση του Microsoft Mail 3.x 
δημιουργεί πρόβλημα, μπορεί να απενεργοποιηθεί βάζοντας ένα ';' μπροστά 
από την αντίστοιχη γραμμή στο Msmail.ini.  Επικοινωνήστε με τον προμηθευτή 
του προϊόντος για περισσότερες πληροφορίες. 

Εάν αντιμετωπίσετε πρόβλημα συσσώρευσης μηνυμάτων στα Εξερχόμενα, 
είναι πιθανό ότι η υπηρεσία πληροφόρησης που προσπαθεί να επεξεργαστεί 
το μήνυμα είναι εσφαλμένη. Για να διορθώσετε το πρόβλημα, κάντε κλικ 
στο μενού "Εργαλεία", στην εντολή "Επιλογές" και στην καρτέλα "Παράδοση" 
και μετά βεβαιωθείτε ότι οι υπηρεσίες πληροφόρησης εμφανίζονται με τη 
σειρά που θέλετε. 

Αφού κάνετε έξοδο από το Microsoft Exchange, ίσως παρατηρήσετε ότι το 
mapisp32 εκτελείται ακόμα. Το Mapisp32 συνεχίζει για περίπου 20 δευτερόλεπτα 
μετά την έξοδό σας από το Microsoft Exchange. 

Οι ηλεκτρονικές φόρμες που δημιουργήθηκαν αυτόματα με το Microsoft Electronic 
Forms Designer δεν ανοίγουν σωστά, όταν έχουν αποθηκευθεί με μορφή αρχείου 
.msg. Μεταφέρετε το αρχείο .msg από το σύστημα αρχείων στα Εισερχόμενα, για να 
ανοίξετε σωστά την ηλεκτρονική φόρμα.

Εάν το κλικ στο "Send" της εντολής "File" του Lotus Approach δεν λειτουργεί, 
βεβαιωθείτε ότι υπάρχει αρχείο Msmail.ini στον κατάλογο {Windows}\.  Εάν το 
αρχείο δεν υπάρχει, απλώς δημιουργήστε ένα κενό αρχείο με αυτό το όνομα. 

Δύο προγράμματα-πελάτες που χρησιμοποιούν Microsoft Mail δεν πρέπει να 
εκτελούνται συγχρόνως προς το ίδιο γραμματοκιβώτιο του Microsoft Mail. 
Αυτή η ρύθμιση δεν υποστηρίζεται.
